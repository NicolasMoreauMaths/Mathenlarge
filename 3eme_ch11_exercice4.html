📋 Exercice 1 — Programme de calcul
Exercice 1
Programme de calcul Développement Factorisation
✓ Vu

On considère le programme de calcul suivant :

1Choisir un nombre
2Mettre ce nombre au carré
3Ajouter le double du nombre de départ
4Ajouter 1
1.Montrer qu'en prenant \(-2\) au départ, on obtient \(1\) à l'arrivée.
2.Montrer qu'en prenant \(x\) au départ, on obtient \(x^2 + 2x + 1\) à l'arrivée.
3.Montrer que ce résultat peut s'écrire \((x+1)(x+1)\).
4.Pour quelle(s) valeur(s) de départ la valeur d'arrivée est-elle égale à \(0\) ?
💡 Indices
Pour la question 1 : applique le programme étape par étape avec le nombre donné.
Pour la question 2 : remplace le nombre particulier par une lettre et applique les mêmes étapes.
Pour la question 3 : développe l'expression proposée et compare-la à ce que tu as obtenu en Q2.
Question 4 : l'expression est un produit égal à zéro. Quelle règle s'applique pour résoudre ce type d'équation ?
Question 1 — Application numérique avec \(-2\)
On applique chaque étape du programme en partant de \(-2\) :
Étape 2 : \((-2)^2 = 4\)
Étape 3 : \(4 + 2 \times (-2) = 4 - 4 = 0\)
Étape 4 : \(0 + 1 =\)
\(1\) ✓
Question 2 — Expression avec \(x\)
On applique chaque étape du programme en partant de \(x\) :
Étape 2 : \(x^2\)
Étape 3 : \(x^2 + 2x\)
Étape 4 :
\(x^2 + 2x + 1\) ✓
Question 3 — Vérification de la forme factorisée
On développe \((x+1)(x+1)\) par double distributivité :
📊 Tableau
×x1
xx²x
1x1
= x² + x + x + 1 = x² + 2x + 1 ✓
✏️ Calcul
= x(x + 1) + 1(x + 1)
= x² + x + x + 1
= x² + 2x + 1 ✓
Identité remarquable
On reconnaît \((a+b)^2 = a^2 + 2ab + b^2\) avec \(a = x\) et \(b = 1\).
Question 4 — Valeur d'arrivée égale à \(0\)
Grâce à la factorisation, l'équation devient :
\(\textcolor{#dc2626}{(x+1)} \times \textcolor{#16a34a}{(x+1)} = 0\)

\(x + 1 = 0\) ou \(x + 1 = 0\)

Facteur = 0
\(x + 1 = 0\)
\(x = -1\)
\(x = -1\)
Les deux facteurs sont identiques, donc la seule valeur d'arrivée nulle est obtenue pour \(\boxed{x = -1}\).
📐 Exercice 2 — Rectangle ABCD
Exercice 2
Géométrie Développement Produit nul
✓ Vu

Dans la figure ci-contre, ABCD est un rectangle :

\(3x - 2\)
\(2x + 5\)
1.Pour quelle valeur de \(x\) le rectangle est-il un carré ? (Dans un carré, la longueur est égale à la largeur.)
2.Montrer que l'aire du rectangle est égale à \(6x^2 + 11x - 10\).
3.Pour quelle(s) valeur(s) de \(x\) l'aire du rectangle est-elle nulle ?
💡 Indices
Question 1 : quelle propriété des dimensions d'un carré te permet d'écrire une équation ?
Question 2 : exprime l'aire du carré en fonction de x, puis développe.
Question 3 : pour quelle valeur de x l'aire est-elle nulle ? Quelle forme a cette équation ?
Question 1 — Condition de carré
Dans un carré, longueur = largeur, donc on résout \(3x - 2 = 2x + 5\) :
Résolution
\(3x - 2 = 2x + 5\)
\(x - 2 = 5\)
\(x = 7\)
\(x = 7\)
Vérification
Pour \(x = 7\) : \(AD = 3 \times 7 - 2 = 19\) et \(AB = 2 \times 7 + 5 = 19\). ✓ C'est bien un carré de côté 19.
Question 2 — Développement de l'aire
On calcule l'aire = \((3x - 2)(2x + 5)\) par double distributivité :
📊 Tableau
×2x5
3x6x²15x
−2−4x−10
= 6x² + 15x − 4x − 10 = 6x² + 11x − 10 ✓
✏️ Calcul
= 3x(2x + 5) + (−2)(2x + 5)
= 6x² + 15x − 4x − 10
= 6x² + 11x − 10 ✓
Question 3 — Aire nulle
Grâce à la factorisation, l'équation aire = 0 devient :
\(\textcolor{#dc2626}{(3x-2)} \times \textcolor{#16a34a}{(2x+5)} = 0\)

\(3x - 2 = 0\) ou \(2x + 5 = 0\)

1er facteur = 0
\(3x - 2 = 0\)
\(3x = 2\)
\(x = \tfrac{2}{3}\)
\(x = \dfrac{2}{3}\)
2e facteur = 0
\(2x + 5 = 0\)
\(2x = -5\)
\(x = -\tfrac{5}{2}\)
\(x = -\dfrac{5}{2}\)
Les solutions de \((3x-2)(2x+5) = 0\) sont \(\boxed{x=\dfrac{2}{3}}\) et \(\boxed{x=-\dfrac{5}{2}}\).
⚠️ Remarque géométrique
\(x = -\frac{5}{2}\) donnerait une longueur négative, ce qui n'a pas de sens géométrique. Dans un contexte de mesure, seule la solution \(x = \frac{2}{3}\) est acceptable.

💡 Indices
Question 1 : calcule chaque branche du programme séparément puis additionne les résultats.
Question 2 : quel facteur est présent dans chacun des deux termes ?
Question 3 : l'expression factorisée est un produit. Quelle règle permet de résoudre ce type d'équation ?
Question 1 — Expression obtenue par l'organigramme
On suit les deux branches à partir de \(x\) :
Branche gauche : \(x^2 \times 3 = 3x^2\)
Branche droite : \(x \times 12 = 12x\)
Résultat :
\(3x^2 + 12x\) ✓
Question 2 — Factoriser \(3x^2 + 12x\)
On identifie \(3x\) comme facteur commun :
\(3x^2 + 12x = \textcolor{#16a34a}{3x} \cdot x + \textcolor{#16a34a}{3x} \cdot 4\)
\(3x^2 + 12x = 3x(x + 4)\)
Question 3 — Résoudre \(3x^2 + 12x = 0\)
Grâce à la factorisation, l'équation devient :
\(\textcolor{#dc2626}{3x} \times \textcolor{#16a34a}{(x+4)} = 0\)

\(3x = 0\) ou \(x + 4 = 0\)

1er facteur = 0
\(3x = 0\)
\(x = 0\)
\(x = 0\)
2e facteur = 0
\(x + 4 = 0\)
\(x = -4\)
\(x = -4\)
Les solutions de \(3x^2 + 12x = 0\) sont \(\boxed{x = 0}\) et \(\boxed{x = -4}\).

⭐ Exercice 4 — Identité remarquable
Exercice 4
a² − b² Factorisation Équation x² = a
✓ Vu
🔍 Rappel : \(a^2 - b^2 = (a+b)(a-b)\). Cette identité transforme une différence de carrés en produit !
1.Factoriser l'expression \(x^2 - 81\).
2.Résoudre l'équation \(x^2 - 81 = 0\) en utilisant l'identité remarquable.
3.Résoudre l'équation \(x^2 - 81 = 0\) en utilisant le format d'équation \(x^2 = a\).
💡 Indices
Question 1 : reconnais la forme de cette expression. Peut-on l'écrire comme une différence de deux carrés ?
Question 2 : l'expression factorisée est un produit égal à zéro. Comment résoudre ?
Question 3 : sans passer par la factorisation, si \(x^2\) est égal à un nombre positif, combien de solutions attend-on ?
Question 1 — Factoriser \(x^2 - 81\)
\(x^2 - 81 = x^2 - 9^2\) — on reconnaît \(a^2 - b^2\) avec \(\textcolor{#dc2626}{a = x}\) et \(\textcolor{#16a34a}{b = 9}\) D'après l'identité \(a^2 - b^2 = (a+b)(a-b)\) : \((x + 9)(x - 9)\)
Question 2 — Résolution par l'identité remarquable
Grâce à la factorisation : \(\textcolor{#dc2626}{(x+9)} \times \textcolor{#16a34a}{(x-9)} = 0\)

\(x + 9 = 0\) ou \(x - 9 = 0\)

1er facteur = 0
\(x + 9 = 0\)
\(x = -9\)
\(x = -9\)
2e facteur = 0
\(x - 9 = 0\)
\(x = 9\)
\(x = 9\)
Les solutions de \((x+9)(x-9) = 0\) sont \(\boxed{x = -9}\) et \(\boxed{x = 9}\).
Question 3 — Résolution par la méthode \(x^2 = a\)
On isole \(x^2\) :
\(x^2 - 81 = 0 \Rightarrow x^2 = 81\)
\(81 > 0\) donc deux solutions. On prend la racine carrée des deux membres :
Solution positive
\(x = \sqrt{81}\)
car \(9^2 = 81\)
\(x = 9\)
Solution négative
\(x = -\sqrt{81}\)
car \((-9)^2 = 81\)
\(x = -9\)
Les solutions de \(x^2 = 81\) sont \(\boxed{x = 9}\) et \(\boxed{x = -9}\).
✓ Cohérence
Les deux méthodes donnent bien les mêmes solutions : \(x = 9\) et \(x = -9\).
