3ème 8 exercices Variables · Double distributivité · Produit nul
📖 Rappels — Les trois outils de la fiche
📦 Variable = mémoire
Une variable est une boîte dans l'ordinateur. Elle a un nom et stocke une valeur.
Le programme peut lire la boîte et la mettre à jour à chaque étape.
Ex : mettre x à x * 3 → on relit x, on calcule, on remet dans x.
📐 Double distributivité
Chaque terme du 1er facteur distribue sur chaque terme du 2e :
(a + b)(c + d) = ac + ad + bc + bd
4 produits → on rassemble les termes en x.
🔑 Produit nul
Si A × B = 0, alors A = 0 ou B = 0.
→ On pose chaque facteur égal à 0, on résout chaque petite équation avec des flèches.
L'équation a alors deux solutions (en général).
🧠 Logique mémoire étape par étape
① Stocker la réponse : mettre x à réponse
② Calculer en relisant : mettre x à x + 5
③ Afficher : dire x
À chaque étape, la boîte x change de contenu. C'est ce qui fait avancer le programme.

Exercice 5 / 8

Produit nul · Facteurs simples
✓ Vu
Résoudre les équations suivantes :
\( \textcolor{#dc2626}{(x+4)} \times \textcolor{#2563eb}{(x-7)} = 0 \)
\( \textcolor{#dc2626}{(3x-6)} \times \textcolor{#2563eb}{(x+2)} = 0 \)
💡 Indices
Si un produit est nul, que peut-on dire de ses facteurs ? Pose une équation pour chacun.
Pour résoudre une équation du type ax + b = 0, isole x en deux étapes : d'abord isoler le terme en x, puis isoler x.
Si \(\textcolor{#dc2626}{A} \times \textcolor{#2563eb}{B} = 0\), alors \(\textcolor{#dc2626}{A = 0}\) ou \(\textcolor{#2563eb}{B = 0}\)
a) \((x+4)(x-7) = 0\)

\(x + 4 = 0\)   ou   \(x - 7 = 0\)

1er facteur = 0
\(x + 4 = 0\)
\(x = -4\)
\(x = -4\)
2e facteur = 0
\(x - 7 = 0\)
\(x = 7\)
\(x = 7\)
Les solutions de \((x+4)(x-7)=0\) sont \(x = -4\) et \(x = 7\).
b) \((3x-6)(x+2) = 0\)

\(3x - 6 = 0\)   ou   \(x + 2 = 0\)

1er facteur = 0
\(3x - 6 = 0\)
\(3x = 6\)
\(x = 2\)
\(x = 2\)
2e facteur = 0
\(x + 2 = 0\)
\(x = -2\)
\(x = -2\)
Les solutions de \((3x-6)(x+2)=0\) sont \(x = 2\) et \(x = -2\).
