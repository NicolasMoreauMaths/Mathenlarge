Chapitre · Repérage — 4ème

Repérage dans l'espace : le pavé droit

Coordonnées 3D · Abscisse · Ordonnée · Cote

I — Le repère de l'espace
1. Définition
📘 Définition

Tout point M d'un pavé droit peut être repéré à partir d'un sommet et des arêtes partant de ce sommet. Un point M est repéré par trois nombres, appelés les coordonnées de M :

M(xM ; yM ; zM)
  • xM est l'abscisse de M — lue sur l'axe rouge (Dx)
  • yM est l'ordonnée de M — lue sur l'axe vert (Dy)
  • zM est la cote (ou altitude) de M — lue sur l'axe bleu (Dz)
2. Les trois axes
x
Axe (Dx) — rouge

Horizontal, allant vers la gauche. Donne l'abscisse.

y
Axe (Dy) — vert

Horizontal, allant vers l'avant. Donne l'ordonnée.

z
Axe (Dz) — bleu

Vertical, allant vers le haut. Donne la cote.

💡 Convention

L'origine du repère est le point O, sommet du pavé droit d'où partent les trois axes. Ses coordonnées sont O(0 ; 0 ; 0).

3. Exemple — Le repère du pavé droit
🔍 Illustration — Repère de l'espace

Dans le repère ci-contre, H est l'origine O(0 ; 0 ; 0). Les trois axes partent de H :

  • (Dx) en rouge est l'axe des abscisses
  • (Dy) en vert est l'axe des ordonnées
  • (Dz) en bleu est l'axe des cotes
Lire les coordonnées d'un point :
→ Je lis sa position sur l'axe rouge (x)
→ Je lis sa position sur l'axe vert (y)
→ Je lis sa hauteur sur l'axe bleu (z)
Exemple : C se trouve en x = 0, y = 0, z = 3 → C(0 ; 0 ; 3)
4. Méthode — Lire les coordonnées d'un point
📐 Méthode — 3 étapes
① Lire l'abscisse x Depuis le point, se déplacer parallèlement aux axes y et z jusqu'à l'axe rouge. La valeur lue est x.
② Lire l'ordonnée y Depuis le point, se déplacer parallèlement aux axes x et z jusqu'à l'axe vert. La valeur lue est y.
③ Lire la cote z Depuis le point, se déplacer parallèlement aux axes x et y jusqu'à l'axe bleu vertical. La valeur lue est z.
📗 Propriété — Points remarquables
  • Un point sur l'axe des abscisses a y = 0 et z = 0.
  • Un point sur l'axe des ordonnées a x = 0 et z = 0.
  • Un point sur l'axe des cotes a x = 0 et y = 0.
  • Un point dans le plan du sol (plan xOy) a z = 0.

J(3 ; 2 ; 0)
→ x = 3 : avancer de 3 sur l'axe rouge

→ z = 0 : rester dans le plan du sol

→ z = 3 : monter de 3 (hauteur max.)
III — Ce que je dois savoir
📦 Repère de l'espace Un sommet O (origine) et trois axes perpendiculaires : (Dx) rouge, (Dy) vert, (Dz) bleu.
📖 Coordonnées d'un point M(x ; y ; z) : x = abscisse (rouge), y = ordonnée (vert), z = cote/altitude (bleu).
🔢 Lire les coordonnées Projeter le point sur chaque axe : parallèlement aux deux autres axes.
✏️ Placer un point Avancer de x sur l'axe rouge, puis de y sur l'axe vert, puis monter de z.
⚠️ Points dans le sol Tout point avec z = 0 est dans le plan du sol (plan xOy).
⚠️ Erreurs fréquentes
  • Inverser x et y : toujours lire x en premier (axe rouge), y en second (axe vert), z en troisième (axe bleu).
  • Oublier z : même si z = 0, il faut toujours écrire la troisième coordonnée. On écrit G(4 ; 0 ; 0) et non G(4 ; 0).
  • Confondre l'axe vert et le rouge : l'axe vert (y) part vers l'avant du pavé, l'axe rouge (x) part vers la gauche.
