Exercice 1 / 6

Substituer
✓ Vu
🔢 Remplacer la lettre par le nombre donné et calculer le résultat.
\(2x + 9\)  avec  \(x = 3\)
\(x^2 - 4\)  avec  \(x = 5\)
\(4a - 7\)  avec  \(a = 3\)
\(x^2 + 2x\)  avec  \(x = 4\)
\(3x - 2y\)  avec  \(x = 4,\; y = 3\)
\(5a + b\)  avec  \(a = 2,\; b = 7\)

💡 Indice

Substituer, c'est remplacer chaque lettre par le nombre donné, partout où elle apparaît. Pense à mettre le nombre entre parenthèses quand il remplace une variable au carré : \(x^2\) avec \(x=5\) donne \(5^2\), pas \(52\).
\(2x + 9\) avec \(x = 3\)
\(= 2 \times 3 + 9 = 6 + 9\)
\(= 15\)
\(x^2 - 4\) avec \(x = 5\)
\(= 5^2 - 4 = 25 - 4\)
\(= 21\)
\(4a - 7\) avec \(a = 3\)
\(= 4 \times 3 - 7 = 12 - 7\)
\(= 5\)
\(x^2 + 2x\) avec \(x = 4\)
\(= 4^2 + 2 \times 4 = 16 + 8\)
\(= 24\)
\(3x - 2y\) avec \(x=4, y=3\)
\(= 3\times4 - 2\times3 = 12 - 6\)
\(= 6\)
\(5a + b\) avec \(a=2, b=7\)
\(= 5\times2 + 7 = 10 + 7\)
\(= 17\)

Exercice 2 / 6

Tester Technique gauche / droite
✓ Vu
⚖️ Pour chaque égalité, tester avec la valeur donnée en calculant séparément le membre gauche et le membre droit, puis conclure.
1.Tester \(5x + 3 = 2x + 12\) pour \(x = 3\).
2.Tester \(x^2 - 1 = (x-1)(x+1)\) pour \(x = 4\).
3.Tester \(3(x - 2) = 3x - 6\) pour \(x = 7\).

💡 Indice

Pour tester une égalité : substitue la valeur dans chaque membre séparément. Que peut-on conclure si les deux résultats coïncident ? Et si on testait une autre valeur, la conclusion resterait-elle forcément valable ?
Égalité 1 — \(5x + 3 = 2x + 12\) pour \(x = 3\)
Membre gauche
\(5x + 3\)
\(= 5 \times 3 + 3\)
\(= 15 + 3\)
\(= 18\)
Membre droit
\(2x + 12\)
\(= 2 \times 3 + 12\)
\(= 6 + 12\)
\(= 18\)
\(18 = 18\) — L'égalité est vraie pour \(x = 3\). ✓

💡 Attention : elle n'est vraie que pour \(x = 3\). Pour \(x = 1\) par exemple : \(8 \neq 14\).

Égalité 2 — \(x^2 - 1 = (x-1)(x+1)\) pour \(x = 4\)
Membre gauche
\(x^2 - 1\)
\(= 4^2 - 1\)
\(= 16 - 1\)
\(= 15\)
Membre droit
\((x-1)(x+1)\)
\(= (4-1)(4+1)\)
\(= 3 \times 5\)
\(= 15\)
\(15 = 15\) — L'égalité est vraie pour \(x = 4\). ✓
Égalité 3 — \(3(x - 2) = 3x - 6\) pour \(x = 7\)
Membre gauche
\(3(x - 2)\)
\(= 3 \times (7 - 2)\)
\(= 3 \times 5\)
\(= 15\)
Membre droit
\(3x - 6\)
\(= 3 \times 7 - 6\)
\(= 21 - 6\)
\(= 15\)
\(15 = 15\) — L'égalité est vraie pour \(x = 7\). ✓

💡 Cette égalité est vraie pour toutes les valeurs de \(x\) : c'est la distributivité simple. On peut le vérifier en développant \(3(x-2) = 3x - 6\).

Exercice 3 / 6

Substituer Réduire d'abord
✓ Vu
✏️ Réduire chaque expression, puis calculer sa valeur pour la valeur donnée. Vérifier que les deux méthodes donnent le même résultat.
1.\(4x - x + 6\)  —  réduire, puis calculer pour \(x = 5\).
2.\(2a + 9 - 3a + 1\)  —  réduire, puis calculer pour \(a = 4\).
3.\(3x^2 + 5x - 2x^2 - 2x\)  —  réduire, puis calculer pour \(x = 2\).\

💡 Indices

Commence par regrouper les termes qui ont la même partie littérale : les termes en \(x\) ensemble, les termes en \(x^2\) ensemble, les constantes ensemble.
Pour vérifier : substitue la valeur directement dans l'expression non réduite. Si le résultat est différent de celui trouvé avec l'expression réduite, c'est qu'il y a une erreur de réduction.
Expression 1
\(\textcolor{#dc2626}{4x} - \textcolor{#dc2626}{x} + 6\)
Réduite : \(3x + 6\)
Pour \(x = 5\) :
\(3 \times 5 + 6 = 15 + 6\)
\(= 21\)
Vérif. : \(4\times5 - 5 + 6 = 20-5+6 = 21\) ✓
Expression 2
\(\textcolor{#dc2626}{2a} - \textcolor{#dc2626}{3a} + \textcolor{#2563eb}{9} + \textcolor{#2563eb}{1}\)
Réduite : \(-a + 10\)
Pour \(a = 4\) :
\(-4 + 10\)
\(= 6\)
Vérif. : \(8+9-12+1 = 6\) ✓
Expression 3
\(\textcolor{#dc2626}{3x^2} - \textcolor{#dc2626}{2x^2} + \textcolor{#2563eb}{5x} - \textcolor{#2563eb}{2x}\)
Réduite : \(x^2 + 3x\)
Pour \(x = 2\) :
\(2^2 + 3\times2 = 4 + 6\)
\(= 10\)
Vérif. : \(12+10-8-4 = 10\) ✓

Exercice 4 / 6

Tester Développer
✓ Vu
⚖️ Tester chaque égalité en substituant la valeur donnée, puis développer le membre gauche pour expliquer pourquoi l'égalité est vraie ou fausse en général.
1.Tester \(4(x + 2) = 4x + 8\) pour \(x = 3\). Puis développer le membre gauche.
2.Tester \(5(x - 1) = 5x - 4\) pour \(x = 6\). Puis développer le membre gauche.

💡 Indices

Commence par substituer la valeur pour obtenir un résultat numérique dans chaque membre. Ce test te dit si l'égalité est vraie pour cette valeur, mais pas encore pour toutes les valeurs.
Tester une valeur ne suffit pas à conclure pour toutes les valeurs. Comment pourrait-on comparer les deux membres en tant qu'expressions algébriques, sans substituer de nombre ?
Égalité 1 — \(4(x + 2) = 4x + 8\) pour \(x = 3\)
Membre gauche
\(4(x + 2)\)
\(= 4 \times (3 + 2)\)
\(= 4 \times 5\)
\(= 20\)
Membre droit
\(4x + 8\)
\(= 4 \times 3 + 8\)
\(= 12 + 8\)
\(= 20\)
\(20 = 20\) — Vraie pour \(x = 3\). ✓
Développement : \(4(x + 2) = \textcolor{#dc2626}{4} \times x + \textcolor{#dc2626}{4} \times 2 = 4x + 8\)
→ Les deux membres sont identiques comme expressions : l'égalité est vraie pour toute valeur de \(x\).
Égalité 2 — \(5(x - 1) = 5x - 4\) pour \(x = 6\)
Membre gauche
\(5(x - 1)\)
\(= 5 \times (6 - 1)\)
\(= 5 \times 5\)
\(= 25\)
Membre droit
\(5x - 4\)
\(= 5 \times 6 - 4\)
\(= 30 - 4\)
\(= 26\)
\(25 \neq 26\) — Fausse pour \(x = 6\).
Développement : \(5(x - 1) = 5x - 5\)
Or le membre droit vaut \(5x - 4\), et \(5x - 5 \neq 5x - 4\) (les constantes diffèrent : \(-5\) ≠ \(-4\)).
→ L'égalité est fausse pour toute valeur de \(x\).

Exercice 5 / 6

Substituer Programme de calcul Double distributivité
✓ Vu
⚙️ On donne deux programmes de calcul. Tester avec \(x = 2\), puis exprimer et comparer les résultats.
Programme A
1Choisir un nombre \(x\)
2Ajouter 4
3Multiplier le résultat par la somme du nombre de départ et 1
Programme B
1Choisir un nombre \(x\)
2Ajouter 2
3Multiplier le résultat par la somme du nombre de départ et 3
1.Appliquer les deux programmes avec \(x = 2\). Les résultats sont-ils égaux ?
2.Exprimer le résultat de chaque programme en fonction de \(x\), puis développer et réduire.
3.Les deux programmes donnent-ils toujours le même résultat ? Justifier.

💡 Indices

Pour la question 2 : le programme A donne un produit de deux expressions en \(x\). Traduis les étapes en langage algébrique : quelle est l'expression à l'étape 2 ? Quelle est celle à l'étape 3 ?
Une fois les deux expressions développées et réduites, compare les termes en \(x^2\), les termes en \(x\), et les constantes. Sont-ils tous identiques ?
Question 1 — Test avec \(x = 2\)
Programme A
\(2 \xrightarrow{+4} 6 \xrightarrow{\times(2+1)}\)
\(= 6 \times 3\)
\(= 18\)
Programme B
\(2 \xrightarrow{+2} 4 \xrightarrow{\times(2+3)}\)
\(= 4 \times 5\)
\(= 20\)
\(18 \neq 20\) — Les deux programmes donnent des résultats différents pour \(x = 2\).
Question 2 — Expression en fonction de \(x\)
Programme A :
Étape 2 : \(x + 4\)  ·  Étape 3 : multiplier par \((x + 1)\)
\(A = (\textcolor{#dc2626}{x+4})(\textcolor{#2563eb}{x+1})\)
📊 Tableau
×\(x\)\(1\)
\(x\)\(x^2\)\(x\)
\(4\)\(4x\)\(4\)
\(A = x^2 + x + 4x + 4 = x^2 + 5x + 4\)
✏️ Calcul
\(\textcolor{#dc2626}{x}(\textcolor{#2563eb}{x+1}) + \textcolor{#dc2626}{4}(\textcolor{#2563eb}{x+1})\)
\(= x^2 + x + 4x + 4\)
\(= x^2 + 5x + 4\)
Programme B :
Étape 2 : \(x + 2\)  ·  Étape 3 : multiplier par \((x + 3)\)
\(B = (\textcolor{#dc2626}{x+2})(\textcolor{#2563eb}{x+3})\)
📊 Tableau
×\(x\)\(3\)
\(x\)\(x^2\)\(3x\)
\(2\)\(2x\)\(6\)
\(B = x^2 + 3x + 2x + 6 = x^2 + 5x + 6\)
✏️ Calcul
\(\textcolor{#dc2626}{x}(\textcolor{#2563eb}{x+3}) + \textcolor{#dc2626}{2}(\textcolor{#2563eb}{x+3})\)
\(= x^2 + 3x + 2x + 6\)
\(= x^2 + 5x + 6\)
Question 3 — Toujours égaux ?
\(A = \) \(x^2 + 5x\) \(+ 4\)  et  \(B = \) \(x^2 + 5x\) \(+ 6\)
Les termes en \(x^2\) et en \(x\) sont identiques, mais les constantes diffèrent : \(+4 \neq +6\).
→ Les deux programmes donnent des résultats toujours différents de 2 (\(B = A + 2\) pour toute valeur de \(x\)).

💡 On vérifie avec \(x = 2\) : \(A = 18\), \(B = 20\), bien \(B - A = 2\). ✓

Exercice 6 / 6

⭐ Bilan Aire & double distributivité
✓ Vu
🌟 Un rectangle a pour longueur \((x + 5)\) cm et pour largeur \((x + 2)\) cm.
1.Calculer l'aire pour \(x = 3\) en substituant directement dans les expressions de longueur et de largeur.
2.Développer et réduire l'expression de l'aire \(\mathcal{A} = (x+5)(x+2)\) en fonction de \(x\).
3.Substituer \(x = 3\) dans l'expression réduite et vérifier que le résultat est le même qu'en question 1.
4.Tester si l'égalité \((x+5)(x+2) = x^2 + 7x + 9\) est vraie pour \(x = 3\). Conclure.

💡 Indices

Pour la question 2, tu dois effectuer une double distributivité : chaque terme de \((x+5)\) se multiplie avec chaque terme de \((x+2)\). Tu obtiendras 4 produits à additionner puis à réduire.
Pour la question 4 : choisir une valeur de \(x\) et substituer dans les deux expressions peut aider à trancher. Mais que faudrait-il vérifier en plus pour être certain que l'égalité est toujours vraie ou toujours fausse ?
Question 1 — Calcul direct pour \(x = 3\)
Longueur
\(x + 5\)
\(= 3 + 5\)
\(= 8 \text{ cm}\)
Largeur
\(x + 2\)
\(= 3 + 2\)
\(= 5 \text{ cm}\)
\(\mathcal{A} = 8 \times 5\)
\(\mathcal{A} = 40 \text{ cm}^2\)
Question 2 — Développer \((x+5)(x+2)\)
📊 Tableau
×\(x\)\(2\)
\(x\)\(x^2\)\(2x\)
\(5\)\(5x\)\(10\)
\(x^2 + 2x + 5x + 10 = x^2 + 7x + 10\)
✏️ Calcul
\(\textcolor{#dc2626}{x}(\textcolor{#2563eb}{x+2}) + \textcolor{#dc2626}{5}(\textcolor{#2563eb}{x+2})\)
\(= x^2 + 2x + 5x + 10\)
\(= x^2 + 7x + 10\)
Question 3 — Substitution \(x = 3\) dans l'expression réduite
\(\mathcal{A} = 3^2 + 7 \times 3 + 10 = 9 + 21 + 10\)
\(\mathcal{A} = 40 \text{ cm}^2\) ✓

💡 Les deux méthodes donnent bien 40 cm² : développer puis substituer est plus rapide quand on doit calculer plusieurs fois !

Question 4 — Tester \((x+5)(x+2) = x^2 + 7x + 9\) pour \(x = 3\)
Membre gauche
\((x+5)(x+2)\)
\(= (3+5)(3+2)\)
\(= 8 \times 5\)
\(= 40\)
Membre droit
\(x^2 + 7x + 9\)
\(= 9 + 21 + 9\)
\(= 39\)
\(40 \neq 39\) — L'égalité est fausse pour \(x = 3\).
On a montré que \((x+5)(x+2) = \) \(x^2 + 7x + \mathbf{10}\), or le membre droit vaut \(x^2 + 7x + \mathbf{9}\).
Les constantes diffèrent (\(+10\) ≠ \(+9\)) : l'égalité est fausse pour toute valeur de \(x\).
