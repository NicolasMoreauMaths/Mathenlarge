🎯 Ce que tu vas travailler

Cette feuille combine les deux types de repérage : la droite graduée (lire et placer des abscisses) et le plan (lire et placer des coordonnées). Les trois derniers exercices sont des problèmes en contexte où tu devras mobiliser ces compétences pour résoudre une situation concrète.
Abscisse
Position d'un point sur la droite graduée. Positive à droite de 0, négative à gauche.
Coordonnées (x ; y)
x = abscisse (horizontal)  ·  y = ordonnée (vertical).
Origine O
Point de coordonnées (0 ; 0) ou d'abscisse 0. Point de référence du repère.
Problème
Situation concrète (distances, cartes, jeux…) où le repérage sert à répondre à une question.

Exercice 1 / 9

📏 Droite graduée Lire des abscisses Entiers
✓ Vu
📏 Droite graduée. Chaque graduation représente 1 unité. Cinq points sont placés ci-dessous.
1.Lis l'abscisse de chacun des cinq points A, B, C, D, E.
2.Quel est le point le plus proche de l'origine ? Justifie.
3.Range les points dans l'ordre décroissant de leurs abscisses.

💡 Indices progressifs

Un point à gauche du 0 a une abscisse négative ; un point à droite a une abscisse positive.
Pour lire l'abscisse, compte le nombre de graduations qui séparent le point du 0, puis applique le signe selon le côté.
Le point le plus proche du 0 est celui dont la distance au 0 est la plus petite. Comment comparer des abscisses des deux côtés du 0 ?

Question 1 — Abscisses

Point A : 4 grad. à gauche → −4

Point B : 2 grad. à droite → 2

Point C : 2 grad. à gauche → −2

Point D : 4 grad. à droite → 4

Point E : 6 grad. à gauche → −6

Question 2 — Point le plus proche du 0

Distances au 0 : A : 4  |  B : 2  |  C : 2  |  D : 4  |  E : 6

Les points B et C sont les plus proches du 0, à 2 graduations chacun.

Question 3 — Ordre décroissant

Ordre décroissant : D (4) > B (2) > C (−2) > A (−4) > E (−6)

Exercice 2 / 9

📐 Plan Lire des coordonnées Entiers
✓ Vu
📐 Repère orthogonal. Chaque carreau vaut 1 unité. Cinq points sont placés dans le repère.
1.Donne les coordonnées (x ; y) de chacun des cinq points.
2.Quel(s) point(s) se trouvent dans le quadrant où x < 0 et y < 0 ?
3.Quel point est le plus haut dans le repère ? Justifie avec son ordonnée.

💡 Indices progressifs

Pour lire les coordonnées d'un point, projette-toi sur chacun des deux axes. Lequel donne l'abscisse, lequel donne l'ordonnée ?
Rappelle-toi ce que signifient les signes pour la position d'un point dans le repère.
Le point le plus haut est celui dont la coordonnée verticale est la plus grande.

Question 1 — Coordonnées

Point A : (2 ; 3)  |  Point B : (−3 ; 1)  |  Point C : (−1 ; −2)

Point D : (3 ; −3)  |  Point E : (4 ; 2)

Question 2 — Quadrant x < 0 et y < 0

Seul le point C (−1 ; −2) vérifie x < 0 et y < 0 : il est dans le troisième quadrant (en bas à gauche).

Question 3 — Point le plus haut

Le point A a l'ordonnée la plus grande (y = 3). C'est donc A qui est le plus haut dans le repère.

Exercice 3 / 9

📏 Droite graduée Fractions — demies
✓ Vu
📏 Droite graduée. Chaque grande graduation vaut 1 unité ; une petite graduation entre deux entiers vaut 12.
1.Donne l'abscisse exacte (sous forme de fraction ou de nombre décimal) de chacun des points A, B, C, D.
2.Encadre chaque abscisse entre deux entiers consécutifs.
3.Range ces quatre abscisses dans l'ordre croissant.

💡 Indices progressifs

Chaque grande graduation est un entier. La petite graduation entre deux entiers est exactement au milieu : elle représente la demi-unité.
Réfléchis à la valeur de la petite graduation par rapport aux entiers qui l'entourent. Comment l'exprimer en fraction ?
Encadrer entre deux entiers : trouve les deux entiers consécutifs qui entourent l'abscisse du point.

Question 1 — Abscisses exactes

Point A : entre −2 et −1, à la demi-graduation → −32 = −1,5

Point B : entre 0 et 1, à la demi-graduation → 12 = 0,5

Point C : entre −1 et 0, à la demi-graduation → −12 = −0,5

Point D : entre 1 et 2, à la demi-graduation → 32 = 1,5

Question 2 — Encadrements

A : −2 < −3/2 < −1  ·  B : 0 < 1/2 < 1  ·  C : −1 < −1/2 < 0  ·  D : 1 < 3/2 < 2

Question 3 — Ordre croissant

A (−3/2) < C (−1/2) < B (1/2) < D (3/2)

Exercice 5 / 9

📏 Droite graduée Fractions — tiers Comparer
✓ Vu
📏 Droite graduée. Chaque grande graduation vaut 1 unité ; entre deux entiers il y a deux petites graduations qui divisent l'unité en 3 parts égales.
1.Donne l'abscisse exacte de chacun des points A, B, C, D sous forme de fraction.
2.Compare les abscisses de A et C. Laquelle est la plus grande ?
3.Place sur la droite le point E d'abscisse 53. Entre quelles graduations se trouve-t-il ?

💡 Indices progressifs

Entre deux entiers consécutifs, il y a 2 petites graduations, donc 3 intervalles égaux. Chaque intervalle vaut 1/3.
Pour les négatifs, le principe est le même que pour les positifs — adapte simplement le signe.
Si une abscisse est supérieure à 1, réfléchis à entre quels entiers elle se situe avant de la placer.

Question 1 — Abscisses

Point A : 2 tiers après 0 → 23

Point B : entre −2 et −1, à 2 tiers depuis −2 → −43

Point C : 1 tiers après 0 → 13

Point D : 2 tiers avant 0 → −23

Question 2 — Comparaison A et C

2/3 > 1/3 donc A > C. L'abscisse de A est la plus grande.

Question 3 — Point E = 5/3

5/3 = 1 + 2/3, donc E est entre 1 et 2, à la 2ème petite graduation après 1.

1.Donne les coordonnées exactes (fraction ou demi-entier) de chacun des six points P, Q, R, S, T, U.
2.Quels points sont dans le demi-plan supérieur (y > 0) ? Quels points sont dans le demi-plan inférieur (y < 0) ?

💡 Indices progressifs

La petite graduation est à mi-chemin entre deux entiers. Quelle fraction cela représente-t-il ?
Pour chaque coordonnée, projette le point sur l'axe correspondant. Si la projection tombe sur une petite graduation, exprime-la en fraction.
Pour identifier le demi-plan, regarde le signe de l'ordonnée du point.

Question 1 — Coordonnées

P (3/2 ; 1)  ·  Q (−1 ; 3/2)  ·  R (−5/2 ; −1)

S (1/2 ; −5/2)  ·  T (2 ; −1/2)  ·  U (−3/2 ; 2)

Question 2 — Demi-plans

Demi-plan supérieur (y > 0) : P (y=1), Q (y=3/2), U (y=2) → au-dessus de l'axe x.

Demi-plan inférieur (y < 0) : R (y=−1), S (y=−5/2), T (y=−1/2) → en dessous de l'axe x.

Problème 7 / 9

📏 Droite graduée 🔍 Problème Distances
✓ Vu
🎯 La course de billes. Sur un terrain de jeu, une allée rectiligne est représentée par une droite graduée. L'origine O correspond à la ligne de départ. Chaque unité représente 1 mètre.

Quatre billes sont lancées depuis O. Après le lancement, elles se retrouvent aux positions suivantes :
Bille A : abscisse −3  ·  Bille B : abscisse 5  ·  Bille C : abscisse −2  ·  Bille D : abscisse 4.
1.Quelle bille a parcouru la plus grande distance depuis la ligne de départ ? Explique en utilisant le vocabulaire de la droite graduée.
2.Deux billes roulent dans la même direction (même signe d'abscisse). Lesquelles ? Quelle est la distance entre elles ?
3.Les billes A et D vont maintenant l'une vers l'autre à la même vitesse. Elles se rencontrent au milieu. Quelle est l'abscisse du point de rencontre ? (Tu peux utiliser la droite graduée pour t'aider.)

💡 Indices progressifs

La distance depuis O ne dépend pas du signe de l'abscisse. Comment appelle-t-on cette notion ?
Pour savoir si deux billes roulent dans le même sens, compare leurs positions relatives par rapport au 0.
Le milieu entre deux points est équidistant des deux. Quelle formule permet de le calculer ?

Question 1 — Distance maximale

Distances depuis O : A : 3 m  |  B : 5 m  |  C : 2 m  |  D : 4 m

La bille B a parcouru la plus grande distance (5 m) : son abscisse est 5, à 5 graduations du 0 vers la droite.

Question 2 — Même direction et distance

Abscisses positives (à droite) : B (5) et D (4). Abscisses négatives (à gauche) : A (−3) et C (−2).

Billes B et D roulent dans la même direction (vers la droite). Distance entre elles : 5 − 4 = 1 m.

Billes A et C roulent aussi dans la même direction (vers la gauche). Distance entre elles : 3 − 2 = 1 m.

Question 3 — Point de rencontre

Abscisse de A : −3  ·  Abscisse de D : 4

Milieu = (−3 + 4) ÷ 2 = 1 ÷ 2 = 1/2

Les billes A et D se rencontrent au point d'abscisse 1/2 (soit 0,5 m à droite de la ligne de départ).

Problème 8 / 9

📐 Plan 🔍 Problème Carte — Navigation
✓ Vu
🗺️ La carte au trésor. Une carte est représentée par un repère orthogonal où chaque unité correspond à 10 mètres. Le point O est le camp de base.

Lieux importants : La fontaine F(−3 ; 2), le vieux chêne V(4 ; 3), la grotte G(−2 ; −4), le pont P(1 ; −2).

Le message secret dit : « Le trésor est enterré à mi-chemin entre la fontaine et le vieux chêne. »
1.Repère les quatre lieux F, V, G, P sur la carte. Quel lieu se situe le plus loin du camp de base O ? (Donne ses coordonnées et justifie en comparant les positions.)
2.Calcule les coordonnées du trésor T, qui se trouve à mi-chemin entre la fontaine F(−3 ; 2) et le vieux chêne V(4 ; 3). (Indice : utilise la même méthode que le milieu d'un segment.)

💡 Indices progressifs

Pour comparer l'éloignement depuis l'origine, réfléchis à ce que représentent les coordonnées de chaque point.
Le milieu entre deux points se calcule coordonnée par coordonnée.
Pour savoir dans quel quadrant se trouve le trésor, regarde simplement le signe de chacune de ses coordonnées.

Question 1 — Lieu le plus éloigné

F(−3 ; 2), V(4 ; 3), G(−2 ; −4), P(1 ; −2).

En observant les coordonnées : V est à 4 unités sur x et 3 sur y ; G est à 2 unités sur x et 4 sur y.

G(−2 ; −4) semble très éloigné (ordonnée −4) tout comme V(4 ; 3). Sans calcul précis, V et G sont les candidats les plus éloignés du camp de base.

Question 2 — Coordonnées du trésor

F(−3 ; 2) et V(4 ; 3).

xT = (−3 + 4) ÷ 2 = 1 ÷ 2 = 1/2

yT = (2 + 3) ÷ 2 = 5 ÷ 2 = 5/2

Le trésor T a pour coordonnées T(1/2 ; 5/2).

Problème 9 / 9

📏 Droite 📐 Plan 🔍 Problème Combiné
✓ Vu
🎿 La station de ski. Une station de montagne dispose d'un thermomètre (droite graduée) et d'un plan de pistes (repère orthogonal). Chaque unité du plan représente 100 mètres.

Thermomètre : Lundi, la température est de −5 °C. Mardi, elle monte de 8 degrés. Mercredi, elle baisse de 3 degrés.

Plan de pistes : Le télésiège part du bas B(−3 ; −4) et arrive en haut H(3 ; 4). La cafétéria C se situe exactement à mi-chemin entre B et H.
✏️ Sur ton cahier :
① La droite graduée ci-dessus sert de modèle : trace-en une identique (de −6 à 6) pour y placer les températures.
② Reproduis le repère ci-dessous sur ton cahier (axes gradués de −5 à 5, 1 carreau = 1 unité) pour y placer les points B, H et C.
1.Droite graduée. Sur ta droite graduée, place les températures de lundi, mardi et mercredi. Quelle est la température de mercredi ? Quel(s) jour(s) fait-il négatif ?
2.Plan. Sur ton repère, place les points B(−3 ; −4) et H(3 ; 4). Calcule ensuite les coordonnées de la cafétéria C, à mi-chemin entre B et H, puis place-la dans ton repère.
3.Bilan. La cafétéria C est-elle plus proche du bas B ou du haut H de la piste ? Explique. Que remarques-tu de particulier sur les coordonnées de C ?

💡 Indices progressifs

Sur la droite des températures, une hausse ou une baisse se traduit par une addition ou une soustraction à l'abscisse.
Pour placer la cafétéria au milieu du trajet, utilise la formule du milieu appliquée à chaque coordonnée.
Regarde les coordonnées que tu viens de calculer pour C. Ce point a-t-il une position particulière dans le repère ?

Question 1 — Températures

Lundi : −5 °C (abscisse −5)

Mardi : −5 + 8 = 3 °C (abscisse 3)

Mercredi : 3 − 3 = 0 °C (abscisse 0 — on est exactement à 0°C !

Seul lundi est une température négative (−5 °C). Mercredi, il fait exactement 0 °C.

Question 2 — Coordonnées de la cafétéria

B(−3 ; −4) et H(3 ; 4)

x_C = (−3 + 3) ÷ 2 = 0 ÷ 2 = 0

y_C = (−4 + 4) ÷ 2 = 0 ÷ 2 = 0

La cafétéria C a pour coordonnées C(0 ; 0) : c'est l'origine du repère !

Question 3 — Bilan

C est le milieu de [BH], donc elle est exactement à égale distance de B et de H. Elle n'est ni plus proche de B ni de H.

Ce qui est remarquable : C(0 ; 0) coïncide avec l'origine O du repère. Le point de départ du télésiège et son arrivée sont symétriques par rapport au centre du repère !
