📖 Ce qu'il faut savoir
Dans un repère à trois axes, chaque point est repéré par trois coordonnées : (abscisse ; ordonnée ; altitude), notées dans cet ordre.
Le point O est l'origine du repère, là où les trois axes se croisent.
💡 Comment lire les coordonnées d'un point ?
On projette le point sur chacun des trois axes :
→ l'abscisse se lit sur l'axe orange (vers l'avant-gauche),
→ l'ordonnée se lit sur l'axe bleu (vers la droite),
→ l'altitude se lit sur l'axe vert (vers le haut).
