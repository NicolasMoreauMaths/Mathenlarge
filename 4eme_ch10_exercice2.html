Opposés de parenthèses

Expressions littérales — 4ème

+ ( ) sans changement − ( ) opposés 4 exercices
📖 Rappel — Supprimer une parenthèse
➕ Signe + devant
On supprime le + et les parenthèses.
Tous les signes à l'intérieur restent inchangés.

\(+(a - b) = a - b\)
➖ Signe − devant
On supprime le − et les parenthèses.
Tous les signes à l'intérieur sont inversés.

\(-(a - b) = -a + b\)
⚠️ Attention : le signe − devant une parenthèse change tous les signes à l'intérieur, pas seulement le premier terme.

Exercice 1 / 4

+ devant − devant
✓ Vu
✏️ Supprimer les parenthèses dans chaque expression.
\(+(3x + 5)\)
\(+(2a - 7)\)
\(-(4x + 3)\)
\(-(5a - 2)\)
\(-(-x + 6)\)
\(-(-3a - 4)\)

💡 Indice

Que se passe-t-il aux signes des termes à l'intérieur d'une parenthèse quand on la supprime ? Est-ce que ça dépend du signe qui est devant la parenthèse ?
Pour \(-(-x + 6)\) : quel est le signe devant la parenthèse ? Que fait-il à chacun des termes à l'intérieur ?
+\((3x + 5)\)
→ signes inchangés
\(= 3x + 5\)
+\((2a - 7)\)
→ signes inchangés
\(= 2a - 7\)
−\((4x + 3)\)
→ tous les signes s'inversent
\(= -4x - 3\)
−\((5a - 2)\)
→ tous les signes s'inversent
\(= -5a + 2\)
−\((-x + 6)\)
→ tous les signes s'inversent
\(= x - 6\)
−\((-3a - 4)\)
→ tous les signes s'inversent
\(= 3a + 4\)

💡 Avec un − devant, c'est comme multiplier chaque terme par −1 : \((+)\) devient \((−)\) et \((−)\) devient \((+)\).

Exercice 2 / 4

− devant puis réduire
✓ Vu
⚖️ Supprimer les parenthèses, puis réduire l'expression obtenue.
\(5x - (2x + 3)\)
\(8a - (3a - 4)\)
\(3x + (x - 5) - 2\)
\(10 - (4x - 1) + 2x\)

💡 Indice

Dans quel ordre faut-il traiter les deux étapes : supprimer les parenthèses, puis regrouper les termes semblables ? Peut-on les faire dans l'ordre inverse ?
Pour \(8a - (3a - 4)\) : quel est le signe devant la parenthèse ? Combien de termes sont à l'intérieur, et que devient le signe de chacun ?
\(5x - (2x + 3)\)
\(= 5x\) \(- 2x - 3\)
\(= 3x - 3\)
\(8a - (3a - 4)\)
\(= 8a\) \(- 3a + 4\)
\(= 5a + 4\)
\(3x + (x - 5) - 2\)
\(= 3x + x - 5 - 2\)
\(= 4x - 7\)
\(10 - (4x - 1) + 2x\)
\(= 10\) \(- 4x + 1\) \(+ 2x\)
\(= -2x + 11\)

💡 Les termes surlignés en rouge sont ceux dont le signe a été inversé par le − devant la parenthèse.

Exercice 3 / 4

+ et − Deux parenthèses
✓ Vu
✏️ Supprimer les deux paires de parenthèses, puis réduire.
\((3x + 1) - (x + 4)\)
\((5a - 2) - (2a - 6)\)
\((2x + 7) + (x - 3)\)
\((6x - 1) - (4x + 5)\)

💡 Indice

Quand il n'y a rien d'écrit devant une parenthèse, quel signe implicite y a-t-il ? Et pour la deuxième parenthèse, quel signe est explicitement indiqué ?
Pour \((5a - 2) - (2a - 6)\) : traite chaque parenthèse séparément selon son signe. Que devient chaque terme une fois les parenthèses supprimées ?
\((3x + 1) - (x + 4)\)
\(= 3x + 1\) \(- x - 4\)
\(= 2x - 3\)
\((5a - 2) - (2a - 6)\)
\(= 5a - 2\) \(- 2a + 6\)
\(= 3a + 4\)
\((2x + 7) + (x - 3)\)
\(= 2x + 7 + x - 3\)
\(= 3x + 4\)
\((6x - 1) - (4x + 5)\)
\(= 6x - 1\) \(- 4x - 5\)
\(= 2x - 6\)

Exercice 4 / 4

⭐ Bilan Périmètre
✓ Vu
🌟 Un grand rectangle a un périmètre de \(P_1 = (6x + 4)\) cm. On enlève un petit rectangle dont le périmètre est \(P_2 = (2x + 8)\) cm.
1.Écrire la différence des périmètres : \(P_1 - P_2\).
2.Supprimer les parenthèses, puis réduire.
3.Calculer le résultat pour \(x = 3\).

💡 Indice

\(P_1 - P_2\) : comment écrire cette soustraction de polynômes avec des parenthèses ? Quel signe se trouve alors devant la deuxième parenthèse ?
Une fois les parenthèses supprimées, quels termes peut-on regrouper entre eux ?
Question 1 — Différence des périmètres
\(P_1 - P_2 = (6x + 4) - (2x + 8)\)
Question 2 — Supprimer les parenthèses
\((6x + 4) - (2x + 8)\)
\(= 6x + 4\) \(- 2x - 8\)
Termes en \(x\) : \(6x - 2x\) · Constantes : \(4 - 8\)
\(= 4x - 4\)
Question 3 — Pour \(x = 3\)
\(4 \times 3 - 4 = 12 - 4\)
\(= 8 \text{ cm}\)
