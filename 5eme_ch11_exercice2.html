Exercice 2 / 8

Tester une égalité

Tester l'égalité suivante en remplaçant x par 2 :

7x + 12 = 19x

Gauche :

7x + 12
= 7 × 2 + 12
= 14 + 12
= 26

Droite :

19x
= 19 × 2
= 38
Donc : Gauche ≠ Droite, l'égalité est fausse.
✓ Vu

Exercice 4 / 8

Tester en développant

Tester l'égalité suivante en développant de chacun des côtés du signe égal :

(x + 3) × 7 = 6(x + 4) − x − 3

🔍 Indices – débloque-les un par un si tu bloques (pour l'exercice 4)

As-tu identifié toutes les parenthèses à développer dans cette équation ? Combien y en a-t-il en tout ?
Après avoir développé, ton expression est-elle vraiment réduite ? Regarde si certains termes peuvent encore être regroupés.
Tu as maintenant une équation sans parenthèses. Que fais-tu habituellement pour résoudre ce type d'équation ?

Gauche :

(x + 3) × 7
= x × 7 + 3 × 7
= 7x + 21

Droite :

6(x + 4) − x − 3
= 6x + 24 − x − 3
= 5x + 21
Donc : Gauche ≠ Droite (7x + 21 ≠ 5x + 21), l'égalité est fausse.
✓ Vu

A = 7(x + 3)

D = a(b + c)

= 7 × x + 7 × 3
= 7x + 21

Exercice 8 / 8

Programmes de calcul

On considère les deux programmes de calculs suivants :

Programme A

- Choisir un nombre
- Multiplier ce nombre par 2
- Ajouter 6 au résultat

Programme B

- Choisir un nombre
- Ajouter 3 à ce nombre
- Multiplier le résultat par 2

1. Prendre 5 comme nombre de départ et appliquer les deux programmes pour donner les valeurs d'arrivées.

2. Montrer que les deux programmes sont toujours égaux, peu importe la valeur de départ.

Question 1 : Avec le nombre 5

Programme A :
• Choisir un nombre : 5
• Multiplier par 2 : 5 × 2 = 10
• Ajouter 6 : 10 + 6 = 16

Programme B :
• Choisir un nombre : 5
• Ajouter 3 : 5 + 3 = 8
• Multiplier par 2 : 8 × 2 = 16

Les deux programmes donnent 16 comme résultat.
Question 2 : Avec le nombre x

Programme A :
• Choisir un nombre : x
• Multiplier par 2 : 2x
• Ajouter 6 : 2x + 6

Programme B :
• Choisir un nombre : x
• Ajouter 3 : x + 3
• Multiplier par 2 : 2(x + 3) = 2x + 6 = 2x + 6

Programme A donne 2x + 6
Programme B donne 2x + 6

✓ Les deux programmes donnent toujours le même résultat : 2x + 6
