Expressions littérales — 4ème

Facteur commun Mettre en évidence 4 exercices
📖 Rappel — Factoriser une expression
Étape 1 — Trouver le facteur commun
Chercher ce qui est commun à tous les termes : un nombre, une lettre, ou les deux.

Dans \(4x + 4\), le facteur commun est \(4\).
Dans \(x^2 + 2x\), le facteur commun est \(x\).
Étape 2 — Mettre en facteur
On écrit le facteur commun devant une parenthèse contenant ce qu'il reste de chaque terme.

\(4x + 4 = 4(x + 1)\)
\(x^2 + 2x = x(x + 2)\)
\(ab + ac = a(b + c)\)  —  on factorise par \(a\)
⚠️ Vérification : développer la réponse doit redonner l'expression de départ.

Exercice 1 / 4

Facteur numérique visible
✓ Vu
🔍 Le facteur commun est un nombre qui divise tous les termes. Factoriser chaque expression.
\(4x + 4\)
\(2x - 4\)
\(6a + 9\)
\(5x - 10\)
\(3x + 12\)
\(8a - 4\)

💡 Indice

Pour trouver le facteur commun, cherche le plus grand entier qui divise tous les coefficients. Comment listes-tu les diviseurs d'un nombre pour trouver ceux qu'il partage avec un autre ?
Une fois le facteur commun trouvé, comment le « mettre devant » la parenthèse ? Que reste-t-il dans la parenthèse pour chaque terme ? Comment vérifier que la factorisation est correcte ?
\(4x + 4\)
\(= \)\(4\)\( \times x + \)\(4\)\( \times 1\)
\(= 4(x + 1)\)
\(2x - 4\)
\(= \)\(2\)\( \times x - \)\(2\)\( \times 2\)
\(= 2(x - 2)\)
\(6a + 9\)
\(= \)\(3\)\( \times 2a + \)\(3\)\( \times 3\)
\(= 3(2a + 3)\)
\(5x - 10\)
\(= \)\(5\)\( \times x - \)\(5\)\( \times 2\)
\(= 5(x - 2)\)
\(3x + 12\)
\(= \)\(3\)\( \times x + \)\(3\)\( \times 4\)
\(= 3(x + 4)\)
\(8a - 4\)
\(= \)\(4\)\( \times 2a - \)\(4\)\( \times 1\)
\(= 4(2a - 1)\)

💡 En orange : le facteur commun extrait. Vérification rapide : redévelopper doit redonner l'expression de départ.

Exercice 2 / 4

Facteur lettre ou nombre
✓ Vu
🔤 Le facteur commun peut être une lettre (ex : \(x^2+2x\)) ou un nombre (ex : \(2x+8\)). Bien regarder chaque terme avant de décider.
\(x^2 + 2x\)
\(3x + x^2\)
\(2x + 8\)
\(5a + a^2\)
\(x^2 - 7x\)
\(4x + x^2\)

💡 Indice

Pour factoriser par une lettre, pose-toi une question : est-ce que cette lettre apparaît dans chaque terme de l'expression ? Que peut-on écrire \(x^2\) comme produit ?
Pour \(2x + 8\) : la lettre n'est pas le facteur commun ici. Y a-t-il un nombre qui divise à la fois 2 et 8 ?
\(x^2 + 2x\)
\(= \)\(x\)\( \times x + \)\(x\)\( \times 2\)
\(= x(x + 2)\)
\(3x + x^2\)
\(= \)\(x\)\( \times 3 + \)\(x\)\( \times x\)
\(= x(3 + x)\)
\(2x + 8\)
\(= \)\(2\)\( \times x + \)\(2\)\( \times 4\)
\(= 2(x + 4)\)
\(5a + a^2\)
\(= \)\(a\)\( \times 5 + \)\(a\)\( \times a\)
\(= a(5 + a)\)
\(x^2 - 7x\)
\(= \)\(x\)\( \times x - \)\(x\)\( \times 7\)
\(= x(x - 7)\)
\(4x + x^2\)
\(= \)\(x\)\( \times 4 + \)\(x\)\( \times x\)
\(= x(4 + x)\)

Exercice 3 / 4

Nombre ou lettre ? Choisir le bon facteur
✓ Vu
🎯 Certaines expressions ont un facteur numérique, d'autres un facteur lettre. À toi d'identifier le bon facteur commun pour chacune.
\(7x + 7\)
\(x^2 + 5x\)
\(6x - 9\)
\(a^2 - a\)
\(4x + 2\)
\(x^2 - x\)

💡 Indice

Pour chaque expression, pose-toi deux questions : (1) La lettre est-elle présente dans tous les termes ? (2) Y a-t-il un nombre qui divise tous les coefficients ?
Pour \(a^2 - a\) : que vaut \(a^2\) sous forme de produit ? Et \(a\) ? Quel facteur commun vois-tu ? Attention : quand un terme est exactement le facteur commun, que reste-t-il dans la parenthèse à sa place ?
\(7x + 7\)
Facteur : \(7\)
\(= 7(x + 1)\)
\(x^2 + 5x\)
Facteur : \(x\)
\(= x(x + 5)\)
\(6x - 9\)
Facteur : \(3\)
\(= 3(2x - 3)\)
\(a^2 - a\)
Facteur : \(a\)
\(a^2 - a = a \times a - a \times 1\)
\(= a(a - 1)\)
\(4x + 2\)
Facteur : \(2\)
\(= 2(2x + 1)\)
\(x^2 - x\)
Facteur : \(x\)
\(x^2 - x = x \times x - x \times 1\)
\(= x(x - 1)\)

💡 Quand un terme est exactement le facteur commun (ex : \(a\) dans \(a^2 - a\)), il reste \(1\) dans la parenthèse — ne pas l'oublier !

Exercice 4 / 4

⭐ Bilan 🔴 Difficile Facteur mixte (nb × lettre)
✓ Vu
🧩 Le facteur commun est ici à la fois un nombre et une lettre. Il faut trouver le plus grand facteur commun possible.
\(3x^2 + 6x\)
\(4x^2 - 8x\)
\(24x^2 + 8x\)
\(10x - 5x^2\)

💡 Indices

Pour \(3x^2 + 6x\) : cherche d'abord un facteur numérique commun aux coefficients, puis demande-toi si la lettre est présente dans chaque terme. Peut-on factoriser par les deux à la fois ?
Quand le facteur commun est à la fois un nombre et une lettre, comment l'écrire ? Que reste-t-il dans chaque terme une fois ce facteur mis en évidence ?
Pour \(24x^2 + 8x\) : quel est le plus grand diviseur commun de 24 et 8 ? La lettre est-elle présente dans les deux termes ? Comment combiner ces deux informations pour former le facteur commun ?
Méthode pour trouver le facteur mixte
On cherche : (1) le plus grand nombre qui divise tous les coefficients, (2) la lettre commune à tous les termes. Le facteur commun est leur produit.
\(3x^2 + 6x\)
3 divise 6 et 3 · \(x\) dans les deux termes
Facteur : \(3x\)
\(= 3x \times x + 3x \times 2\)
\(= 3x(x + 2)\)
Vérif : \(3x(x+2) = 3x^2+6x\) ✓
\(4x^2 - 8x\)
4 divise 8 et 4 · \(x\) dans les deux termes
Facteur : \(4x\)
\(= 4x \times x - 4x \times 2\)
\(= 4x(x - 2)\)
Vérif : \(4x(x-2) = 4x^2-8x\) ✓
\(24x^2 + 8x\)
8 divise 24 et 8 · \(x\) dans les deux termes
Facteur : \(8x\)
\(= 8x \times 3x + 8x \times 1\)
\(= 8x(3x + 1)\)
Vérif : \(8x(3x+1) = 24x^2+8x\) ✓
\(10x - 5x^2\)
5 divise 10 et 5 · \(x\) dans les deux termes
Facteur : \(5x\)
\(= 5x \times 2 - 5x \times x\)
\(= 5x(2 - x)\)
Vérif : \(5x(2-x) = 10x-5x^2\) ✓
