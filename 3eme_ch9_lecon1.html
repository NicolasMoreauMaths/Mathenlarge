Chapitre

Fonctions linéaires et fonctions affines

Vocabulaire indispensable

Avant de commencer, voici les deux mots-clés que l'on retrouvera dans toute la leçon. Pour une fonction affine \(f(x) = ax + b\) :

a Coefficient directeur

C'est le nombre qui multiplie \(x\). Il indique la pente de la droite : combien monte-t-on (ou descend-on) quand on avance d'une unité vers la droite.

b Ordonnée à l'origine

C'est le nombre ajouté à \(ax\). Il indique la valeur de \(f(0)\), c'est-à-dire le point où la droite coupe l'axe des ordonnées.

\(f(x) = \underbrace{a}_{\text{coeff. directeur}} \times x \;+\; \underbrace{b}_{\text{ord. à l'origine}}\)
I. Fonctions linéaires
1) Définition
📘 Définition

Une fonction linéaire \(f\) est une fonction qui, à un nombre \(x\), fait correspondre le nombre \(a \times x\) où \(a\) est un nombre fixé.

\(f : x \mapsto ax \quad\) ou \(\quad f(x) = ax\)

C'est une fonction affine avec \(b = 0\) : la droite passe toujours par l'origine.

✏️ Exemple

La fonction \(f(x) = 3x\) est linéaire, \(a = 3\). On a \(f(0) = 0\), \(f(1) = 3\), \(f(-2) = -6\).

📗 Propriété — Proportionnalité

La fonction linéaire \(f : x \mapsto ax\) représente une situation de proportionnalité de coefficient \(a\).

2) Représentation graphique
📗 Propriété

La représentation graphique d'une fonction linéaire \(f : x \mapsto ax\) est une droite passant par l'origine \(O(0\,;\,0)\).

📋 Méthode — Tracer la droite
  • 1Placer l'ordonnée à l'origine — pour une fonction linéaire, c'est l'origine \(O(0\,;\,0)\)
  • 2Avancer d'1 unité vers la droite en abscisse
  • 3Monter de \(a\) unités si \(a > 0\), descendre de \(|a|\) unités si \(a < 0\)
  • 4Relier les deux points par une droite
✏️ Exemple — \(g(x) = -2x\)

\(a = -2\), \(b = 0\). On part de \(O(0\,;\,0)\), on avance de 1, on descend de 2.

\(g(x) = -2x\) — a = −2, b = 0
● b = 0 → départ en O(0 ; 0) → +1 en abscisse ↓ a = −2 → descendre de 2
II. Fonctions affines
1) Définition
📘 Définition

Une fonction affine \(f\) est une fonction qui, à un nombre \(x\), fait correspondre \(a \times x + b\) où \(a\) et \(b\) sont deux nombres fixés.

\(f : x \mapsto ax + b \quad\) ou \(\quad f(x) = ax + b\)
  • \(a\) est le coefficient directeur
  • \(b\) est l'ordonnée à l'origine
✏️ Exemple

La fonction \(f(x) = 2x + 1\) est affine avec \(a = 2\) et \(b = 1\). On a \(f(0) = 1\), \(f(3) = 7\), \(f(-1) = -1\).

💡 Cas particuliers
Si \(b = 0\)

La fonction \(f(x) = ax\) est une fonction linéaire. La droite passe par l'origine.

Si \(a = 0\)

La fonction \(f(x) = b\) est une fonction constante. La droite est horizontale.

📗 Propriété

Une fonction affine modélise une situation où il y a proportionnalité entre les accroissements. Quand \(x\) augmente de 1, \(f(x)\) augmente de \(a\).

2) Représentation graphique
📗 Propriété

La représentation graphique d'une fonction affine \(f : x \mapsto ax + b\) est une droite de coefficient directeur \(a\) et d'ordonnée à l'origine \(b\).

📋 Méthode — Tracer la droite
  • 1Placer l'ordonnée à l'origine \((0\,;\,b)\)
  • 2Avancer d'1 unité vers la droite en abscisse
  • 3Monter de \(a\) unités si \(a > 0\), descendre de \(|a|\) unités si \(a < 0\)
  • 4Relier les deux points par une droite
✏️ Exemple
Représenter les fonctions \(g(x) = 2x + 1\) et \(h(x) = -2x + 3\).
\(g(x) = 2x + 1\) — \(a=2,\;b=1\)
● b = 1 → départ en (0 ; 1) → +1 en abscisse ↑ a = +2 → monter de 2
\(h(x) = -2x + 3\) — \(a=-2,\;b=3\)
● b = 3 → départ en (0 ; 3) → +1 en abscisse ↓ a = −2 → descendre de 2

\(g(x) = 2x + 1\)

Point 1\(g(0) = 1\) → \((0\,;\,1)\)
Point 2\(g(1) = 3\) → \((1\,;\,3)\)

\(h(x) = -2x + 3\)

Point 1\(h(0) = 3\) → \((0\,;\,3)\)
Point 2\(h(1) = 1\) → \((1\,;\,1)\)
💡 Remarques — Sens de variation
↗ Si \(a > 0\)
La droite est croissante.
↘ Si \(a < 0\)
La droite est décroissante.
→ Si \(a = 0\)
La droite est horizontale.
