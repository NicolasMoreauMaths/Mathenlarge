📖 Rappels – Les variables en Scratch
📦 Une variable = une mémoire
Imagine une petite boîte dans l'ordinateur. Cette boîte a un nom (par exemple x) et elle peut stocker une information : un nombre, un texte…
Le programme peut lire cette boîte (pour s'en souvenir) et modifier son contenu à tout moment.
Exemple : mettre x à 5 → la boîte x contient maintenant 5.
🔄 Réutiliser la mémoire pour avancer
Pour faire avancer un calcul, on relit la boîte et on la met à jour :
mettre x à x * 3 → on prend ce qui est dans x, on multiplie par 3, et on remet le résultat dans x.
ajouter 5 à x → on rajoute 5 à ce qui est déjà dans x.
⚠️ mettre = remplace tout · ajouter = s'ajoute à ce qui existe déjà
🧠 Comment fonctionne un programme avec une variable ?
Étape 1 : On crée la mémoire → mettre x à réponse (x stocke le nombre choisi par l'utilisateur)
Étape 2 : On utilise la mémoire pour calculer → mettre x à x * 4 (on relit x, on calcule, on remet dans x)
Étape 3 : On relit la mémoire pour afficher → dire x (on affiche ce que x contient)
À chaque étape, la boîte x change de contenu. C'est ça qui fait "avancer" le programme !
