Coordonnées dans un repère — 5ème

📐 Repère ✓ Abscisse & Ordonnée 5ème

📌 Vocabulaire essentiel

Un repère est formé de deux droites graduées perpendiculaires. Pour situer un point dans le plan, on utilise deux nombres : ses coordonnées.

L'origine
Le point O situé à l'intersection des deux axes. Ses coordonnées sont toujours (0 ; 0).
L'abscisse
La première coordonnée d'un point : elle se lit sur l'axe horizontal (axe des abscisses, noté x).
L'ordonnée
La deuxième coordonnée d'un point : elle se lit sur l'axe vertical (axe des ordonnées, noté y).

On note les coordonnées d'un point M ainsi : M(abscisse ; ordonnée), par exemple M(2 ; −3). Attention à l'ordre : abscisse en premier, ordonnée en second !

Question 1 — Valeur d'une subdivision

Entre deux grandes graduations consécutives, il y a 2 petits intervalles pour une distance de 1 unité.

Chaque petit intervalle vaut 1 ÷ 2 = 12 unité.

Question 2 — Coordonnées

Point L : 1 petite grad. à droite → abscisse = 12 ; 3 petites grad. en haut → ordonnée = 32 → L(12 ; 32)

Point M : 3 petites grad. à gauche → abscisse = −32 ; 2 petites grad. en bas → ordonnée = −1 → M(−32 ; −1)

Point N : 3 petites grad. à droite → abscisse = 32 ; 3 petites grad. en bas → ordonnée = −32 → N(32 ; −32)

Point P : 2 petites grad. à gauche → abscisse = −1 ; 1 petite grad. en haut → ordonnée = 12 → P(−1 ; 12)

Question 3 — Cadran de N

Oui : N a une abscisse positive (3/2 > 0) et une ordonnée négative (−3/2 < 0), donc il se trouve bien dans le cadran en bas à droite.

Question 2 — Nature de la figure

Les points A et B ont la même ordonnée (2) : côté AB horizontal.
