Exercice 1 / 6

Substituer
✓ Vu
🔢 Remplacer la lettre par le nombre donné et calculer le résultat.
\(x - 7\)  avec  \(x = 15\)
\(3x + 2\)  avec  \(x = 4\)
\(2a + 5\)  avec  \(a = 3\)
\(x^2\)  avec  \(x = 5\)
\(3x + 2y\)  avec  \(x = 2,\; y = 4\)
\(4a - b\)  avec  \(a = 3,\; b = 6\)

💡 Indice

Substituer, c'est remplacer une lettre par une valeur. Mais dans une expression comme \(3x + 2\), combien de fois la lettre apparaît-elle ? Et une fois remplacée, quelle opération reste-t-il à faire ?
\(x - 7\) avec \(x = 15\)
\(= 15 - 7\)
\(= 8\)
\(3x + 2\) avec \(x = 4\)
\(= 3 \times 4 + 2 = 12 + 2\)
\(= 14\)
\(2a + 5\) avec \(a = 3\)
\(= 2 \times 3 + 5 = 6 + 5\)
\(= 11\)
\(x^2\) avec \(x = 5\)
\(= 5^2\)
\(= 25\)
\(3x + 2y\) avec \(x=2,\,y=4\)
\(= 3\times2 + 2\times4 = 6 + 8\)
\(= 14\)
\(4a - b\) avec \(a=3,\,b=6\)
\(= 4\times3 - 6 = 12 - 6\)
\(= 6\)

Exercice 2 / 6

Tester une égalité Technique gauche / droite
✓ Vu
⚖️ Pour chaque égalité, tester avec la valeur donnée en calculant séparément le membre gauche et le membre droit, puis conclure.
1.Tester \(7x + 12 = 19x\) pour \(x = 2\).
2.Tester \(x^2 + 5 = 3x + 7\) pour \(x = 3\).
3.Tester \(2(x + 3) = 2x + 6\) pour \(x = 5\).

💡 Indice

Une égalité a deux côtés. Comment vérifier qu'ils valent la même chose pour une valeur donnée ? Que faut-il calculer, et comment conclure ensuite ?
Égalité 1 — \(7x + 12 = 19x\) pour \(x = 2\)
Membre gauche
\(7x + 12\)
\(= 7 \times 2 + 12\)
\(= 14 + 12\)
\(= 26\)
Membre droit
\(19x\)
\(= 19 \times 2\)
\(= 38\)
\(26 \neq 38\) — L'égalité est fausse pour \(x = 2\).
Égalité 2 — \(x^2 + 5 = 3x + 7\) pour \(x = 3\)
Membre gauche
\(x^2 + 5\)
\(= 3^2 + 5\)
\(= 9 + 5\)
\(= 14\)
Membre droit
\(3x + 7\)
\(= 3 \times 3 + 7\)
\(= 9 + 7\)
\(= 16\)
\(14 \neq 16\) — L'égalité est fausse pour \(x = 3\).
Égalité 3 — \(2(x + 3) = 2x + 6\) pour \(x = 5\)
Membre gauche
\(2(x + 3)\)
\(= 2 \times (5 + 3)\)
\(= 2 \times 8\)
\(= 16\)
Membre droit
\(2x + 6\)
\(= 2 \times 5 + 6\)
\(= 10 + 6\)
\(= 16\)
\(16 = 16\) — L'égalité est vraie pour \(x = 5\). ✓

💡 Une égalité vraie pour une valeur n'est pas forcément vraie pour toutes les valeurs. La question 3 illustre la distributivité — elle est vraie pour toute valeur de \(x\).

Exercice 3 / 6

Substituer puis réduire
✓ Vu
✏️ Réduire chaque expression, puis calculer sa valeur pour la valeur donnée. Vérifier que les deux méthodes donnent le même résultat.
1.\(5x - 2x + 3\)  —  réduire, puis calculer pour \(x = 4\).
2.\(3a + 7 - a + 1\)  —  réduire, puis calculer pour \(a = 2\).
3.\(2x^2 - x^2 + 4x\)  —  réduire, puis calculer pour \(x = 3\).

💡 Indice

Avant de substituer une valeur, l'expression est-elle sous sa forme la plus simple ? Est-ce que ça change quelque chose de simplifier d'abord ou de substituer directement ?
Pour vérifier ta réduction, tu as deux versions de la même expression : l'originale et la réduite. Comment les utiliser ensemble pour détecter une erreur éventuelle ?
Expression 1
\(5x - 2x + 3\)
Réduite : \(3x + 3\)
Pour \(x = 4\) :
\(3 \times 4 + 3 = 12 + 3\)
\(= 15\)
Vérif. directe : \(5\times4 - 2\times4 + 3 = 20 - 8 + 3 = 15\) ✓
Expression 2
\(3a + 7 - a + 1\)
Réduite : \(2a + 8\)
Pour \(a = 2\) :
\(2 \times 2 + 8 = 4 + 8\)
\(= 12\)
Vérif. directe : \(3\times2 + 7 - 2 + 1 = 12\) ✓
Expression 3
\(2x^2 - x^2 + 4x\)
Réduite : \(x^2 + 4x\)
Pour \(x = 3\) :
\(3^2 + 4 \times 3 = 9 + 12\)
\(= 21\)
Vérif. directe : \(2\times9 - 9 + 12 = 21\) ✓

Exercice 4 / 6

Tester Développer
✓ Vu
⚖️ Tester chaque égalité en substituant la valeur donnée, puis développer les deux membres pour expliquer pourquoi l'égalité est vraie ou fausse.
1.Tester \(3(x + 4) = 3x + 12\) pour \(x = 5\). Puis développer le membre gauche.
2.Tester \(2(x + 1) = 2x + 3\) pour \(x = 2\). Puis développer le membre gauche.

💡 Indice

Pour tester numériquement, quelle valeur de \(x\) essaierais-tu en premier ? Et si tu voulais savoir si l'égalité est toujours vraie, pas seulement pour une valeur, comment procéder autrement que par substitution ?
Si les membres développés sont identiques en tant qu'expressions littérales, l'égalité est vraie pour toutes les valeurs. Sinon, elle est parfois vraie et parfois fausse selon \(x\).
Égalité 1 — \(3(x + 4) = 3x + 12\) pour \(x = 5\)
Membre gauche
\(3(x + 4)\)
\(= 3 \times (5 + 4)\)
\(= 3 \times 9\)
\(= 27\)
Membre droit
\(3x + 12\)
\(= 3 \times 5 + 12\)
\(= 15 + 12\)
\(= 27\)
\(27 = 27\) — Vraie pour \(x = 5\). ✓
Développement : \(3(x + 4) = 3 \times x + 3 \times 4 = 3x + 12\)
→ Les deux membres sont identiques en tant qu'expressions : l'égalité est vraie pour toute valeur de \(x\).
Égalité 2 — \(2(x + 1) = 2x + 3\) pour \(x = 2\)
Membre gauche
\(2(x + 1)\)
\(= 2 \times (2 + 1)\)
\(= 2 \times 3\)
\(= 6\)
Membre droit
\(2x + 3\)
\(= 2 \times 2 + 3\)
\(= 4 + 3\)
\(= 7\)
\(6 \neq 7\) — Fausse pour \(x = 2\).
Développement : \(2(x + 1) = 2x + 2\)
Or le membre droit vaut \(2x + 3\), et \(2x + 2 \neq 2x + 3\) (les constantes diffèrent).
→ L'égalité est fausse pour toute valeur de \(x\).

Exercice 5 / 6

Substituer Programme de calcul
✓ Vu
⚙️ On donne deux programmes de calcul. Tester avec \(x = 3\), puis exprimer et comparer les résultats.
Programme A
1Choisir un nombre \(x\)
2Multiplier par 2
3Ajouter 6
Programme B
1Choisir un nombre \(x\)
2Ajouter 3
3Multiplier par 2
1.Appliquer les deux programmes avec \(x = 3\). Les résultats sont-ils égaux ?
2.Exprimer le résultat de chaque programme en fonction de \(x\), puis réduire.
3.Les deux programmes donnent-ils toujours le même résultat ? Justifier.

💡 Indices

Pour le programme B, l'ordre des opérations est important : on fait d'abord l'addition, puis la multiplication. Comment traduit-on « ajouter 3 puis multiplier par 2 » en expression littérale, sans sauter l'étape intermédiaire ?
Tu as maintenant les deux expressions réduites. Qu'est-ce qui te permettrait de conclure que deux programmes donnent toujours le même résultat, quelle que soit la valeur choisie ?
Question 1 — Test avec \(x = 3\)
Programme A
\(3 \xrightarrow{\times 2} 6 \xrightarrow{+6}\)
\(= 12\)
Programme B
\(3 \xrightarrow{+3} 6 \xrightarrow{\times 2}\)
\(= 12\)
\(12 = 12\) — Les deux programmes donnent le même résultat pour \(x = 3\). ✓
Question 2 — Expression en fonction de \(x\)
Programme A : \(x \to 2x \to 2x + 6\)
\(A = 2x + 6\)
Programme B : \(x \to x+3 \to 2(x+3) = 2x + 6\)
\(B = 2x + 6\)
Question 3 — Toujours égaux ?
Les deux expressions réduites sont identiques : \(2x + 6 = 2x + 6\).
→ Les deux programmes donnent toujours le même résultat, quelle que soit la valeur de \(x\).

Exercice 6 / 6

⭐ Bilan Périmètre & substitution
✓ Vu
🌟 Un rectangle a une longueur de \(3x + 1\) cm et une largeur de \(x + 2\) cm.
1.Exprimer le périmètre \(P\) en fonction de \(x\), puis le réduire.
2.Substituer \(x = 4\) dans l'expression réduite pour calculer le périmètre.
3.Vérifier en substituant \(x = 4\) directement dans les expressions de longueur et largeur, puis en appliquant la formule du périmètre.

💡 Indices

Pour exprimer le périmètre de ce rectangle, quelles sont ses dimensions en fonction de \(x\) ? Comment combiner ces dimensions pour obtenir le périmètre total ?
Pour la vérification, tu dois remplacer \(x\) par une valeur numérique. Dans quel ordre faut-il procéder : d'abord calculer chaque dimension, puis le périmètre ? Ou peut-on substituer directement dans l'expression réduite ?
Question 1 — Périmètre réduit
\(P = 2(3x+1) + 2(x+2)\)
\(= 6x + 2 + 2x + 4\)
Termes en \(x\) : \(6x + 2x\) · Constantes : \(2 + 4\)
\(P = 8x + 6\) cm
Question 2 — Substitution \(x = 4\)
\(P = 8 \times 4 + 6 = 32 + 6\)
\(P = 38 \text{ cm}\)
Question 3 — Vérification directe
Longueur
\(3x + 1\)
\(= 3 \times 4 + 1\)
\(= 13 \text{ cm}\)
Largeur
\(x + 2\)
\(= 4 + 2\)
\(= 6 \text{ cm}\)
\(P = 2 \times 13 + 2 \times 6 = 26 + 12 = \mathbf{38 \text{ cm}}\) ✓

💡 Les deux méthodes donnent bien \(38\) cm — la substitution dans l'expression réduite est plus rapide !
