I — La droite graduée
1. Définitions et vocabulaire
📘 Définition

Une droite graduée est une droite sur laquelle on a choisi :

  • une origine, notée O, à laquelle on associe le nombre 0 ;
  • un sens positif (conventionnellement vers la droite), matérialisé par une flèche ;
  • une unité de longueur qui permet de placer les nombres entiers.
📘 Définition — Abscisse

À chaque point M d'une droite graduée correspond un unique nombre appelé son abscisse. On note M(a) pour dire que le point M a pour abscisse a.

M(a) : M est le point d'abscisse a

Si a > 0, le point est à droite de l'origine.
Si a < 0, le point est à gauche de l'origine.
Si a = 0, le point est à l'origine.

🔍 Illustration

Droite graduée avec l'unité 1. Les points A, B, C, D, E sont placés.

On lit : A(−4) — B(3) — C(−1) — D(5) — E(−3)

2. Graduations avec fractions
📗 Propriété

Lorsqu'une unité est divisée en n intervalles égaux, chaque petit intervalle vaut 1/n.

Pour lire l'abscisse d'un point :

  1. Compter le nombre de petits intervalles séparant le point de l'origine (ou de la grande graduation la plus proche).
  2. Multiplier ce nombre par la valeur d'un intervalle.
  3. Affecter le signe − si le point est à gauche du 0.
🔍 Illustration — Demi-unités

Chaque unité est découpée en 2 parties égales → chaque petit intervalle vaut ½.

Il y a 2 petits intervalles par unité, donc chaque intervalle vaut ½.
F(−5/2) — G(3/2) — H(−1) — J(1/2) — K(5/2)

3. Méthode : lire une abscisse (étape par étape)
📐 Méthode
① Déterminer l'unité Repérer deux grandes graduations consécutives et vérifier combien de petits intervalles les séparent.
② Calculer la valeur d'un intervalle Valeur = 1 ÷ (nombre de petits intervalles). Si chaque unité est divisée en 4 : chaque intervalle vaut ¼.
③ Compter les intervalles Compter le nombre de petits intervalles du 0 jusqu'au point (ou depuis la grande graduation connue la plus proche).
④ Affecter le signe Point à droite du 0 → abscisse positive. Point à gauche → abscisse négative.
4. Exemple guidé — Abscisses fractionnaires
✏️ Exemple

Sur la droite ci-dessus, détermine l'abscisse du point G (3 demi-unités à droite du 0).

① UnitéChaque unité est divisée en 2 petits intervalles.
② IntervalleValeur d'un petit intervalle = 1 ÷ 2 = ½

II — Le repère du plan
1. Définitions
📘 Définition — Repère du plan

Un repère du plan est formé de :

  • un point O, appelé origine du repère ;
  • une droite graduée horizontale, l'axe des abscisses (axe des x) ;
  • une droite graduée verticale, l'axe des ordonnées (axe des y), perpendiculaire à la première ;
  • les deux axes ont la même unité et se croisent à l'origine O(0 ; 0).

Le sens positif est vers la droite sur l'axe des abscisses et vers le haut sur l'axe des ordonnées.

📘 Définition — Coordonnées

Dans un repère, tout point M est repéré par un couple de nombres (x ; y) :

M(x ; y) où x est l'abscisse et y est l'ordonnée

x se lit sur l'axe horizontal (on lit en premier).
y se lit sur l'axe vertical (on lit en second).

🔍 Illustration — Repère avec 5 points

Repère, unité = 1 carré. Points A, B, C, D, E.

A(2 ; 4) — B(−3 ; 1) — C(4 ; −2) — D(−1 ; −4) — E(0 ; 2)

2. Méthode : lire les coordonnées d'un point
📐 Méthode
① Tracer une perpendiculaire à l'axe des x Depuis le point, trace (mentalement ou au crayon) une ligne verticale jusqu'à l'axe des abscisses. Note la valeur lue : c'est l'abscisse x.
② Tracer une perpendiculaire à l'axe des y Depuis le point, trace une ligne horizontale jusqu'à l'axe des ordonnées. Note la valeur lue : c'est l'ordonnée y.
③ Écrire les coordonnées On écrit toujours (abscisse ; ordonnée), dans cet ordre. Exemple : M(3 ; −2) signifie x = 3 et y = −2.
3. Méthode : placer un point dans le repère
📐 Méthode
① Repérer l'abscisse Partir de O. Se déplacer de x unités sur l'axe horizontal (droite si x > 0, gauche si x < 0). Tracer une ligne verticale depuis ce point.
② Repérer l'ordonnée Depuis O. Se déplacer de y unités sur l'axe vertical (haut si y > 0, bas si y < 0). Tracer une ligne horizontale depuis ce point.
③ Marquer l'intersection Le point cherché se trouve à l'intersection des deux droites tracées. Le nommer et noter ses coordonnées.
4. Cas particuliers importants
📗 Propriété
  • Un point sur l'axe des abscisses a une ordonnée égale à 0 : M(a ; 0).
  • Un point sur l'axe des ordonnées a une abscisse égale à 0 : M(0 ; b).
  • Le point O(0 ; 0) est l'origine du repère : il est à l'intersection des deux axes.
✏️ Exemple

Sur la figure ci-dessus, le point E(0 ; 2) se trouve sur l'axe des ordonnées (abscisse = 0). Que peut-on dire d'un point sur l'axe des abscisses ?

ObservationUn point sur l'axe des abscisses s'y trouve → sa ligne horizontale vers l'axe vertical arrive sur 0.
ConclusionSon ordonnée est 0. On l'écrit M(a ; 0).
À retenir : Axe des abscisses : y = 0. Axe des ordonnées : x = 0. Ces deux propriétés sont symétriques.
III — Les cadrans du repère ⭐ Notion difficile

🗺️ Imagine que le repère est une carte ! Les deux axes (horizontal et vertical) divisent la carte en 4 grandes zones, comme les points cardinaux. On appelle ces zones des cadrans. Le secret pour savoir dans quel cadran est un point ? Il suffit de regarder le signe de ses deux coordonnées.

1. Les deux axes partagent le plan en 4 zones
🔍 Schéma — Les 4 cadrans

L'axe horizontal (rouge) et l'axe vertical (bleu) se croisent en O et forment 4 cadrans, numérotés 1 à 4 dans le sens contraire des aiguilles d'une montre.

2. Comprendre chaque cadran
1 Cadran 1 — En haut à droite x > 0  et  y > 0

➡️ Le point est à droite du 0 (x positif) et au-dessus du 0 (y positif). Les deux coordonnées sont positives.

Exemple : A(3 ; 5) → x = 3 > 0 ✓ et y = 5 > 0 ✓ → Cadran 1

2 Cadran 2 — En haut à gauche x < 0  et  y > 0

⬅️ Le point est à gauche du 0 (x négatif) et au-dessus du 0 (y positif). x est négatif, y est positif.

Exemple : B(−4 ; 2) → x = −4 < 0 ✓ et y = 2 > 0 ✓ → Cadran 2

3 Cadran 3 — En bas à gauche x < 0  et  y < 0

↙️ Le point est à gauche du 0 (x négatif) et en dessous du 0 (y négatif). Les deux coordonnées sont négatives.

Exemple : C(−2 ; −3) → x = −2 < 0 ✓ et y = −3 < 0 ✓ → Cadran 3

4 Cadran 4 — En bas à droite x > 0  et  y < 0

↘️ Le point est à droite du 0 (x positif) et en dessous du 0 (y négatif). x est positif, y est négatif.

Exemple : D(5 ; −1) → x = 5 > 0 ✓ et y = −1 < 0 ✓ → Cadran 4

3. Le cas particulier : les points sur les axes
⚠️ Attention

Un point qui se trouve sur un axe n'appartient à aucun cadran. Pourquoi ? Parce qu'il est sur la frontière entre deux cadrans, pas à l'intérieur d'un seul.

Sur l'axe des abscisses

Son ordonnée y = 0.
Exemple : E(4 ; 0) — sur la droite horizontale.

Sur l'axe des ordonnées

Son abscisse x = 0.
Exemple : F(0 ; −2) — sur la droite verticale.

Et O(0 ; 0), l'origine ? Elle est au croisement des deux axes → elle n'appartient à aucun des 4 cadrans non plus.

4. L'astuce pour ne plus se tromper
📐 Méthode — 2 questions à se poser

Pour trouver le cadran d'un point M(x ; y), pose-toi ces deux questions dans l'ordre :

Question 1 : x est-il positif ou négatif ?

Positif (x > 0) → le point est à droite.  |  Négatif (x < 0) → le point est à gauche.

Question 2 : y est-il positif ou négatif ?

Positif (y > 0) → le point est en haut.  |  Négatif (y < 0) → le point est en bas.

Combine les deux réponses :
Droite + Haut = Cadran 1   |   Gauche + Haut = Cadran 2
Gauche + Bas = Cadran 3   |   Droite + Bas = Cadran 4

Point G : 4 carreaux à gauche, 2 en haut → G(−4 ; 2)

Point H : 2 carreaux à droite, 3 en bas → H(2 ; −3)

Point I : 3 carreaux à gauche, 4 en bas → I(−3 ; −4)

Exercice 6 — Coordonnées fractionnaires dans le plan
✏️ Exercice 6 ⭐

Le repère ci-dessous a des grandes graduations (entiers) et des petites subdivisions. Réponds aux questions.

1. Quelle est la valeur d'une petite subdivision entre deux grandes graduations ?

2. Lis les coordonnées de V, W, X et Y sous forme de fractions ou d'entiers.

3. Quel(s) point(s) ont une abscisse et une ordonnée de même signe ? Dans quel cadran se trouvent-ils ?

Question 1 — Valeur d'une subdivision

Résultat : Entre deux grandes graduations, il y a 2 petits intervalles pour 1 unité → chaque intervalle vaut ½.

Question 2 — Coordonnées

Point V : 3 petites grad. à droite → x = 3/2 ; 2 petites grad. en haut → y = 1 → V(3/2 ; 1)

Point W : 1 petite grad. à gauche → x = −1/2 ; 3 petites grad. en haut → y = 3/2 → W(−1/2 ; 3/2)

Point X : 2 petites grad. à droite → x = 1 ; 3 petites grad. en bas → y = −3/2 → X(1 ; −3/2)

Point Y : 3 petites grad. à gauche → x = −3/2 ; 2 petites grad. en bas → y = −1 → Y(−3/2 ; −1)

Question 3 — Même signe

V(3/2 ; 1) : 3/2 > 0 et 1 > 0 → même signe ✓ → cadran 1 (haut droit)

Y(−3/2 ; −1) : −3/2 < 0 et −1 < 0 → même signe ✓ → cadran 3 (bas gauche)

W : signes opposés (−/+) → cadran 2. X : signes opposés (+/−) → cadran 4.

Réponse : Les points V et Y ont leurs coordonnées de même signe.
V — Récapitulatif du chapitre
📘 Ce que je dois savoir
📏 Droite graduée Origine O, sens positif vers la droite, unité fixée. L'abscisse est la valeur lue sur la droite.
🔢 Lire une abscisse Déterminer la valeur d'un intervalle, compter les intervalles depuis 0, affecter le signe selon le côté.
📐 Repère du plan Deux droites graduées perpendiculaires en O. Tout point est repéré par (abscisse ; ordonnée).
📖 Lire des coordonnées Projeter le point sur chaque axe : la projection sur l'axe horizontal donne x ; la projection sur l'axe vertical donne y.
✏️ Placer un point Se déplacer de x unités horizontalement depuis O, puis de y unités verticalement. Marquer l'intersection.
🗂 Cadrans 4 cadrans déterminés par les signes de x et y. Axe des x : y = 0. Axe des y : x = 0.
⚠️ Erreurs fréquentes
  • Inverser abscisse et ordonnée : on lit toujours x d'abord, puis y. M(3 ; −2) ≠ M(−2 ; 3).
  • Oublier le signe : un point à gauche du 0 a une abscisse négative, un point en bas de 0 a une ordonnée négative.
  • Confondre petites et grandes graduations : vérifier la valeur d'une subdivision avant de lire.
  • Dire qu'un point est dans un cadran alors qu'il est sur un axe : un point sur un axe n'appartient à aucun cadran.
