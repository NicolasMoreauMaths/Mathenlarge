Partie 1 – Additions

Exercice 1 / 7

Additions directes
✓ Vu

Effectuer les calculs suivants :

A = (−7,5) + (−12,5)
B = (+18) + (−27)
C = (−3,6) + (+8,4)
D = (+15) + (−15)
E = (−4,2) + (−3,8) + (+10)
F = (+2,5) + (−6) + (+3,5)

💡 Indices

Imagine une droite numérique : tu pars d'un nombre et tu « marches » d'une certaine distance dans une direction. Que se passe-t-il quand les deux nombres pointent dans la même direction ? Dans des directions opposées ?
Quand les deux nombres ont des signes opposés, essaie sur un exemple concret : (+8) + (−3). En imaginant une droite numérique, à quelle distance du zéro arrives-tu ? De quel côté ? Qu'est-ce qui détermine le signe du résultat ?
Pour les expressions à 3 termes : est-il possible de tout calculer d'un seul coup ? Ou vaut-il mieux avancer pas à pas en notant un résultat intermédiaire ? Quelle stratégie as-tu déjà utilisée pour les exercices à 2 termes ?

Calcul A

A = (−7,5) + (−12,5)

Même signe (−) : 7,5 + 12,5 = 20, on garde −

A = −20

Calcul B

B = (+18) + (−27)

Signes opposés : 27 − 18 = 9, on garde − (27 > 18)

B = −9

Calcul C

C = (−3,6) + (+8,4)

Signes opposés : 8,4 − 3,6 = 4,8, on garde + (8,4 > 3,6)

C = +4,8

Calcul D

D = (+15) + (−15)

Opposés → résultat nul

D = 0

Calcul E

E = (−4,2) + (−3,8) + (+10)

Étape 1 : (−4,2) + (−3,8) = −8

Étape 2 : (−8) + (+10) = +2

E = +2

Calcul F

F = (+2,5) + (−6) + (+3,5)

Étape 1 : (+2,5) + (−6) = −3,5

Étape 2 : (−3,5) + (+3,5) = 0

F = 0

Exercice 2 / 7

Additions avec fractions
✓ Vu

Calculer les expressions suivantes (résultats sous forme de fraction irréductible ou entier) :

A = (−3/4) + (+1/4)
B = (+5/6) + (−4/3)
C = (−7/2) + (+5/4)
D = (+3/5) + (−9/10) + (+1/2)

💡 Indices

Pour additionner des fractions, les dénominateurs doivent être identiques. Cherche d'abord le plus petit commun multiple (PPCM) des dénominateurs.
Pour convertir une fraction en un autre dénominateur, rappelle-toi qu'on peut toujours multiplier numérateur et dénominateur par un même nombre sans changer la valeur. Quel nombre choisir pour que le dénominateur devienne celui que tu cibles ?
Pour D, trois fractions avec trois dénominateurs différents : cherche d'abord un dénominateur commun à 5, 10 et 2. Quel est le plus petit nombre divisible par ces trois valeurs ? Une fois toutes les fractions converties, comment additionnes-tu les numérateurs en tenant compte des signes ?

Calcul A

A = (−3/4) + (+1/4)

Même dénominateur : −3 + 1 = −2

A = −2/4 = −1/2

Calcul B

B = (+5/6) + (−4/3)

Mise au même dénominateur (PPCM = 6) :

B = (+5/6) + (−8/6)

5 − 8 = −3

B = −3/6 = −1/2

Calcul C

C = (−7/2) + (+5/4)

Mise au même dénominateur (PPCM = 4) :

C = (−14/4) + (+5/4)

−14 + 5 = −9

C = −9/4

Calcul D

D = (+3/5) + (−9/10) + (+1/2)

PPCM(5, 10, 2) = 10 :

D = (+6/10) + (−9/10) + (+5/10)

6 − 9 + 5 = 2

D = 2/10 = 1/5

Partie 2 – Soustractions

Exercice 3 / 7

Soustractions
✓ Vu

Effectuer les soustractions suivantes en transformant chaque soustraction en addition :

A = (+14) − (+9)
B = (−6) − (−15)
C = (+3,5) − (−2,5)
D = (−12) − (+7) − (−3)
E = (−4,8) − (−1,2) − (+6)
F = (5/4) − (3/2)

💡 Indices

Soustraire, c'est l'opération inverse d'additionner. Existe-t-il un lien entre soustraire un nombre et additionner quelque chose ? Que donnerait (−5) − (+2) si tu le traduisais en addition ?
Pour B : tu as deux nombres de signes opposés après la transformation. Lequel a la plus grande valeur absolue : 6 ou 15 ? C'est lui qui impose son signe au résultat final.
Pour les calculs à 3 termes : es-tu obligé de tout transformer en une seule fois avant de calculer ? Commence par traiter les deux premières opérations, puis reviens à la troisième une fois que tu as un résultat intermédiaire.

Calcul A

A = (+14) − (+9)
A = (+14) + (−9)
A = +5

Calcul B

B = (−6) − (−15)
B = (−6) + (+15)
B = +9

Calcul C

C = (+3,5) − (−2,5)
C = (+3,5) + (+2,5)
C = +6

Calcul D

D = (−12) − (+7) − (−3)
D = (−12) + (−7) + (+3)

Étape 1 : (−12) + (−7) = −19

Étape 2 : (−19) + (+3) = −16

D = −16

Calcul E

E = (−4,8) − (−1,2) − (+6)
E = (−4,8) + (+1,2) + (−6)

Étape 1 : (−4,8) + (+1,2) = −3,6

Étape 2 : (−3,6) + (−6) = −9,6

E = −9,6

Calcul F

F = (+5/4) − (+3/2)
F = (+5/4) + (−3/2) = (+5/4) + (−6/4)

5 − 6 = −1

F = −1/4

Partie 3 – Calculs sans parenthèses

Exercice 4 / 7

Sans parenthèses
✓ Vu

Simplifier en écriture sans parenthèses, puis calculer :

A = (+8) − (−4) + (−11) − (+2)
B = (−15) − (+3) − (−9) + (+1)
C = −(+6) + (−7) − (−14) − (−2)

💡 Indices

Pour supprimer les parenthèses, réfléchis à ce que le signe devant la parenthèse « fait » au signe à l'intérieur. Essaie sur un cas simple : que devient − (−4) ? Et + (−4) ? Quelle logique observes-tu ?
Concentre-toi sur un terme à la fois. Pour chaque couple (signe devant + parenthèse), applique la règle de l'indice 1. Par exemple : que vaut − avant (−4) ? Et + avant (−11) ? Réécris chaque terme un par un.
Une fois les parenthèses supprimées, tu te retrouves face à une suite de nombres avec leurs signes. Y a-t-il une façon de les organiser pour simplifier le calcul, plutôt que d'additionner dans l'ordre ?

Calcul A

Étape 1 : Supprimer les parenthèses

A = (+8) − (−4) + (−11) − (+2)
A = +8 +4 −11 −2

Étape 2 : Regrouper

A = +8 +4 −11 −2
A = +12 −13
A = −1

Calcul B

Étape 1 : Supprimer les parenthèses

B = (−15) − (+3) − (−9) + (+1)
B = −15 −3 +9 +1

Étape 2 : Regrouper

B = +9 +1 −15 −3
B = +10 −18
B = −8

Calcul C

Étape 1 : Supprimer les parenthèses

C = −(+6) + (−7) − (−14) − (−2)
C = −6 −7 +14 +2

Étape 2 : Regrouper

C = +14 +2 −6 −7
C = +16 −13
C = +3

Exercice 5 / 7

Sans parenthèses – niveau 2
✓ Vu

Calculer en simplifiant d'abord l'écriture (enlever parenthèses et signes + redondants) :

A = −(−5) + (+3) − (+8) − (−6) + (−1)
B = −(+9) − (−4) + (−7) − (−2) − (+3)
C = +(+2) − (−3,5) + (−6,5) − (+1) − (−4)

💡 Indices

Avant de calculer quoi que ce soit, as-tu une version de l'expression sans parenthèses ? C'est souvent plus facile de voir ce qu'on fait une fois que les parenthèses ont disparu. Comment les supprimer ici ?
Une fois sans parenthèses, identifie les termes positifs et négatifs. Additionne-les séparément, puis fais la différence des deux totaux.
Pour C, si tu as un doute sur l'un des termes, prends-le isolément et pose-toi la question : quel signe se retrouve quand je supprime la parenthèse sur ce seul terme ? Traiter les termes un par un peut éviter les erreurs d'accumulation.

Calcul A

A = −(−5) + (+3) − (+8) − (−6) + (−1)
A = +5 +3 −8 +6 −1
A = +5 +3 +6 −8 −1
A = +14 −9
A = +5

Calcul B

B = −(+9) − (−4) + (−7) − (−2) − (+3)
B = −9 +4 −7 +2 −3
B = +4 +2 −9 −7 −3
B = +6 −19
B = −13

Calcul C

C = +(+2) − (−3,5) + (−6,5) − (+1) − (−4)
C = +2 +3,5 −6,5 −1 +4
C = +2 +3,5 +4 −6,5 −1
C = +9,5 −7,5
C = +2

Partie 4 – Sommes algébriques

Exercice 6 / 7

Sommes algébriques
✓ Vu

Calculer les sommes algébriques suivantes en regroupant les termes de même signe :

A = −8 +13 −5 +2 −7
B = +6 −11 −3 +9 −4 +1
C = −1,5 +4 −2,5 +7 −6 +3

💡 Indices

Dans une somme algébrique, chaque terme vient déjà avec son signe. Est-ce qu'il faut les traiter dans l'ordre où ils apparaissent, ou peut-on les réorganiser librement ? Quelle propriété de l'addition permet ça ?
Pour A : identifie d'abord visuellement quels termes sont précédés du signe + et lesquels du signe −. Colorie-les mentalement (ou sur papier) en deux groupes, puis additionne chaque groupe indépendamment avant de conclure.
Pour C avec des décimaux : fais attention à ne pas confondre les valeurs. Additionne séparément les termes positifs (il y en a 3), puis les termes négatifs (il y en a 3 également). La somme de chaque groupe est-elle un entier ou un décimal ?

Somme A

A = −8 +13 −5 +2 −7

Regroupement :

A = +13 +2 −8 −5 −7
A = +15 −20
A = −5

Somme B

B = +6 −11 −3 +9 −4 +1

Regroupement :

B = +6 +9 +1 −11 −3 −4
B = +16 −18
B = −2

Somme C

C = −1,5 +4 −2,5 +7 −6 +3

Regroupement :

C = +4 +7 +3 −1,5 −2,5 −6
C = +14 −10
C = +4

Exercice 7 / 7

Problème – Sommes algébriques
✓ Vu

1. Calculer la somme algébrique S = −14 + 8 − 3 + 17 − 5 − 9 + 6

2. On pose x = −6 et y = +9. Calculer l'expression E = 2x − y + 3x − 2y en substituant d'abord puis en simplifiant.

3. Vrai ou faux ? Justifier : la somme algébrique T = −17 + 9 − 4 + 12 est positive.

💡 Indices

Pour S : avant de calculer dans l'ordre, observe les termes. Y a-t-il des remarques à faire sur leurs valeurs ou leurs signes ? Est-ce qu'une organisation différente rendrait le calcul plus rapide ou plus lisible ?
Pour E, commence par remplacer chaque lettre par sa valeur, sans encore calculer. Que devient 2x quand x = −6 ? Rappelle-toi comment multiplier un entier par un nombre négatif avant de passer à la somme algébrique.
Pour T, calcule la somme en regroupant comme tu l'as appris. Une fois le résultat trouvé, relis attentivement la question : que signifie exactement « être positif » pour un nombre ? Est-ce que cela inclut zéro ?

Question 1 – Calcul de S

S = −14 +8 −3 +17 −5 −9 +6

Regroupement :

S = +8 +17 +6 −14 −3 −5 −9
S = +31 −31
S = 0

Question 2 – Calcul de E

Substitution avec x = −6 et y = +9 :

E = 2×(−6) − (+9) + 3×(−6) − 2×(+9)
E = (−12) − (+9) + (−18) − (+18)

Supprimer les parenthèses :

E = −12 −9 −18 −18

Tous les termes sont négatifs :

E = −57

Question 3 – T est-elle positive ?

T = −17 +9 −4 +12
T = +9 +12 −17 −4
T = +21 −21
T = 0
Conclusion : FAUX — T n'est pas positive. T = 0, donc T est nulle (ni positive, ni négative).
