Racines carrées, cas à distinguer, solutions — 3ème
3ème Algèbre 8 exercices
📐 Fiche méthode — Résoudre \(x^2 = a\)
Pour résoudre \(\textcolor{#dc2626}{x^2} = \textcolor{#2563eb}{a}\), on distingue trois cas selon le signe de \(a\).
La racine carrée \(\sqrt{a}\) est toujours positive — c'est le nombre positif dont le carré vaut \(a\).
❌ Si \(a < 0\)
Un carré est toujours \(\geq 0\).
\(x^2\) ne peut pas être négatif.

→ Aucune solution
Exemple : \(x^2 = -4\) → impossible
🟡 Si \(a = 0\)
\(x^2 = 0\) signifie que \(x\) doit valoir \(0\).
Le carré de \(0\) est \(0\).

→ Une seule solution : \(x = 0\)
Exemple : \(x^2 = 0\) → \(x = 0\)
✅ Si \(a > 0\)
Deux nombres ont leur carré égal à \(a\) :
\(\sqrt{a}\) et \(-\sqrt{a}\)

→ Deux solutions : \(x = \sqrt{a}\) ou \(x = -\sqrt{a}\)
Exemple : \(x^2 = 9\) → \(x=3\) ou \(x=-3\)

Exercice 1 / 7
Très facileIdentifier le cas
✓ Vu
🔍 Avant de calculer, il faut toujours regarder le signe de \(a\) pour savoir combien de solutions attendtre.

Pour chaque équation, indiquer combien de solutions elle possède (0, 1 ou 2), sans calculer les solutions.

a. \(x^2 = 25\)
b. \(x^2 = -9\)
c. \(x^2 = 0\)
d. \(x^2 = 7\)
e. \(x^2 = -1\)
f. \(x^2 = 100\)

💡 Indices

Regarde uniquement le nombre à droite du \(=\) : est-il négatif, nul ou positif ?
Le signe du nombre à droite détermine le nombre de solutions. Rappelle-toi ce que vaut \(x^2\) selon les valeurs de x.
a) \(x^2 = 25\)
On regarde le signe de \(25\).
\(25 > 0\) — le membre de droite est strictement positif.
\(\Rightarrow\) 2 solutions.
b) \(x^2 = -9\)
On regarde le signe de \(-9\).
\(-9 < 0\) — le membre de droite est négatif.
Or, pour tout nombre \(x\), on a toujours \(x^2 \geq 0\).
Un carré ne peut donc jamais valoir un nombre négatif.
\(\Rightarrow\) Aucune solution.
c) \(x^2 = 0\)
On regarde le signe de \(0\).
\(a = 0\) — cas particulier : le seul nombre dont le carré vaut \(0\) est \(0\) lui-même.
En effet : \(0^2 = 0\), et si \(x^2 = 0\) alors \(x \times x = 0\), donc \(x = 0\).
\(\Rightarrow\) 1 solution : \(x = 0\).
d) \(x^2 = 7\)
On regarde le signe de \(7\).
\(7 > 0\) — le membre de droite est strictement positif.
\(\Rightarrow\) 2 solutions.
e) \(x^2 = -1\)
On regarde le signe de \(-1\).
\(-1 < 0\) — le membre de droite est négatif.
Même raisonnement qu'en b) : \(x^2 \geq 0\) pour tout \(x\), donc \(x^2\) ne peut pas valoir \(-1\).
\(\Rightarrow\) Aucune solution.
f) \(x^2 = 100\)
On regarde le signe de \(100\).
\(100 > 0\) — le membre de droite est strictement positif.
\(\Rightarrow\) 2 solutions.
Réponses : a) 2 solutions — b) 0 solution — c) 1 solution — d) 2 solutions — e) 0 solution — f) 2 solutions
Exercice 2 / 7
FacileCarrés parfaits
✓ Vu
💡 Un carré parfait est un entier dont la racine carrée est un entier : \(1, 4, 9, 16, 25, 36, 49, 64, 81, 100\ldots\)

Résoudre les équations suivantes :

\(x^2 = 36\)
\(x^2 = 49\)

💡 Indices

Quel entier multiplié par lui-même donne 36 ? Pense aux tables de multiplication.
Tu as trouvé une valeur positive. Y a-t-il d'autres valeurs de x dont le carré donne le même résultat ?
a) \(x^2 = 36\)
\(36 > 0\) donc deux solutions. On prend la racine carrée des deux membres :
Solution positive
\(x = \sqrt{36}\)
car \(6^2 = 36\)
\(x = 6\)
Solution négative
\(x = -\sqrt{36}\)
car \((-6)^2 = 36\)
\(x = -6\)
Les solutions de \(x^2=36\) sont \(\boxed{{x=6}}\) et \(\boxed{{x=-6}}\).
b) \(x^2 = 49\)
\(49 > 0\) donc deux solutions :
Solution positive
\(x = \sqrt{49}\)
car \(7^2 = 49\)
\(x = 7\)
Solution négative
\(x = -\sqrt{49}\)
car \((-7)^2 = 49\)
\(x = -7\)
Les solutions de \(x^2=49\) sont \(\boxed{{x=7}}\) et \(\boxed{{x=-7}}\).
Exercice 3 / 7
FacileCas particuliers
✓ Vu

Résoudre les équations suivantes :

\(x^2 = -16\)
\(x^2 = 0\)
\(x^2 = -3\)

💡 Indices

Demande-toi : peut-on élever un nombre au carré et obtenir un résultat négatif ? Que vaut \(x^2\) si \(x=0\) ?
Rappelle-toi : peut-on obtenir un nombre négatif en élevant quelque chose au carré ?
a) \(x^2 = -16\)
\(-16 < 0\) — le membre de droite est négatif.
Or, pour tout nombre \(x\), on a toujours \(x^2 \geq 0\).
Un carré ne peut donc jamais valoir un nombre négatif.
\(\Rightarrow\) Aucune solution.
b) \(x^2 = 0\)
\(x^2 = x \times x = 0\).
Un produit est nul si et seulement si l'un de ses facteurs est nul.
Ici les deux facteurs sont \(x\), donc nécessairement \(x = 0\).
\(\Rightarrow\) Une unique solution : \(\boxed{x = 0}\).
c) \(x^2 = -3\)
\(-3 < 0\) — le membre de droite est négatif.
Même raisonnement qu'en a) : \(x^2 \geq 0\) pour tout \(x\), donc \(x^2\) ne peut pas valoir \(-3\).
\(\Rightarrow\) Aucune solution.

Exercice 4 / 7
IntermédiaireRacine non entière
✓ Vu
📌 Quand \(a\) n'est pas un carré parfait, on laisse la réponse sous forme \(\sqrt{a}\) (forme exacte). La valeur approchée à la calculatrice est une approximation, pas la solution exacte.

Résoudre les équations suivantes :

\(x^2 = 7\)
\(x^2 = 3\)

💡 Indices

7 est-il un carré parfait ? Comment écrire exactement les solutions dans ce cas ?
La forme exacte utilise le symbole racine carrée — pas besoin de calculer une valeur décimale approchée.
a) \(x^2 = 7\)
\(7 > 0\) donc deux solutions. \(7\) n'est pas un carré parfait, on laisse sous forme \(\sqrt{7}\) :
Solution positive
\(x = \sqrt{7}\)
\(\approx 2,646\ldots\) (irrationnel)
\(x = \sqrt{7}\)
Solution négative
\(x = -\sqrt{7}\)
\(\approx -2,646\ldots\)
\(x = -\sqrt{7}\)
Les solutions de \(x^2=7\) sont \(\boxed{{x=\sqrt{{7}}}}\) et \(\boxed{{x=-\sqrt{{7}}}}\) (formes exactes).
b) \(x^2 = 3\)
\(3 > 0\) donc deux solutions :
Solution positive
\(x = \sqrt{3}\)
\(\approx 1,732\ldots\)
\(x = \sqrt{3}\)
Solution négative
\(x = -\sqrt{3}\)
\(\approx -1,732\ldots\)
\(x = -\sqrt{3}\)
Les solutions de \(x^2=3\) sont \(\boxed{{x=\sqrt{{3}}}}\) et \(\boxed{{x=-\sqrt{{3}}}}\).

Exercice 6 / 7
DifficileDeux méthodes
✓ Vu
🔗 Connexion : l'identité \(a^2-b^2=(a+b)(a-b)\) permet de relier les équations \(x^2=a\) et les équations produit nul. On peut résoudre par deux chemins différents !

Résoudre \(x^2 - 25 = 0\) de deux façons différentes :

1. En isolant \(x^2\) (méthode \(x^2=a\))
2. En factorisant par \(a^2-b^2=(a+b)(a-b)\) (méthode produit nul)
3. Vérifier que les deux méthodes donnent le même résultat.

💡 Indices

Tu peux résoudre cette équation de deux façons différentes. L'une passe par l'isolation de \(x^2\), l'autre par la reconnaissance d'une identité remarquable.
Avec la méthode du produit nul : une fois factorisé, applique la règle du produit nul. Les deux méthodes doivent donner le même résultat.
Méthode 1 — Isoler \(x^2\)
On ajoute \(\textcolor{#2563eb}{25}\) des deux membres :
\(x^2 = 25\)
\(25 > 0\) → deux solutions : \(\textcolor{#dc2626}{x=5}\) ou \(\textcolor{#16a34a}{x=-5}\)
Méthode 2 — Factoriser avec \(a^2-b^2\)
On reconnaît une différence de carrés avec \(a=x\) et \(b=5\) :
\(x^2 - 25 = \textcolor{#dc2626}{(x+5)}\textcolor{#16a34a}{(x-5)} = 0\)
Produit nul :
\(\textcolor{#dc2626}{x+5=0}\) ou \(\textcolor{#16a34a}{x-5=0}\)
\(\textcolor{#dc2626}{x=-5}\) ou \(\textcolor{#16a34a}{x=5}\)
Vérification
Les deux méthodes donnent bien les mêmes solutions. Vérification : \(5^2=25\) ✓ et \((-5)^2=25\) ✓
Les deux méthodes donnent \(\boxed{{x=5}}\) et \(\boxed{{x=-5}}\).
Exercice 7 / 7
ProblèmeGéométrie
✓ Vu

Un jardinier veut créer deux carrés de gazon de même côté \(x\) cm.

1. L'aire totale des deux carrés est \(98\) cm². Écrire une équation en \(x\).
2. Résoudre cette équation.
3. Quelle est la longueur du côté de chaque carré ? Justifier le choix de la solution.

💡 Indices

Exprime l'aire totale des deux carrés en fonction de x, puis traduis la condition donnée par une équation.
L'équation peut avoir deux solutions mathématiques. Lesquelles sont acceptables dans ce contexte géométrique ?
Question 1 — Mise en équation
L'aire d'un carré de côté \(x\) est \(x^2\). Deux carrés identiques donnent :
\(2x^2 = 98\)
Question 2 — Résolution
On divise les deux membres par \(\textcolor{#2563eb}{2}\) :
\(x^2 = 49\)
\(49 > 0\) → deux solutions :
Solution positive
\(x = \sqrt{49}\)
\(x = 7\)
Solution négative
\(x = -\sqrt{49}\)
\(x = -7\)
Question 3 — Interprétation
Mathématiquement : \(x = 7\) ou \(x = -7\).
Mais \(x\) représente un côté de carré → une longueur est toujours positive.
\(\Rightarrow\) On rejette \(x = -7\) car une longueur ne peut pas être négative.
Le côté de chaque carré mesure \(\boxed{{7}}\) cm. Vérification : \(2 \times 7^2 = 2 \times 49 = 98\) ✓
